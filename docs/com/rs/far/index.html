    三农问题专家、乡村建设的精神内核

    2012-10-15 Atom.X

    温铁军于90年代《中国改革》杂志做农村调研时期，在媒体上奠定了“用脚做学问”的田野调查风格和良好个人IP形象。以杂志社新闻编辑的功绩，获得国家级大奖荣誉。

    温三农 #

    而后得以涉足三农类高教公职编制，享用体制渠道便利和海量资源，并建立三农和乡村建设核心团队，开辟无竞争者的（特立独行）“三农问题专家”专属赛道，江湖人称“温三农”。

    以新闻传媒为工具，加工三农和乡村建设的内容素材，制作爆款新闻话题，挂上教授博士（国家战略）专家等大头衔，忽悠见识少的老农民和涉世未深的年轻学生。

    在2005年后，温三农在高教体系的理论研究基本毫无功绩建树，更无任何“乡村建设”实践成果，满世界找各种新奇理论概念往“三农问题”的大筐里装，以“三农问题”专家的名气，一直躺吃体制福利和政策红利。

    甚至在进入2020年后，越来越昏庸，继续新闻媒体的老套路，经常与其他公知开直播信口开河胡说八道，流量密码是语不惊人死不休，遭到同僚和其他学者集体批判，场面蔚为状况，为何会发生这种匪夷所思的状况？

    这就涉及到传媒、学术、高教部门的腐败和不作为的深层问题，让不学无术德不配位的伪专家持续在体制内外发挥舆论作用。

    这个世界缺提问题的理论专家吗？缺的是解决方案专家，也就是缺实践家，实实在在解决问题的人，那是无数的基层劳动者，包括村书记、村官、乡村创业者、新老村民、乡贤、投资者、文化活动组织者。

    网红大师 #

    温三农本质非学者，而是个网红，在互联网时代叫自媒体创作者，他有新闻传播专业背景，是制造并引导舆论的公知大师，而非具备学术研究素养且负责任的严谨学者。

    认清事实，再强调一遍：“三农和乡村建设”不是温三农的工作，而是其做网红媒体的内容素材。

    “三农和乡村建设”自2000之后成为媒体广泛关注的新词，温铁军作为先行探索者，通过《中国改革》杂志社发动的大学生下乡支农的农村调研开始，定义了多个行业关键词，或创造其独属的专有名词，掌握三农话语权，包括“三农（问题）专家、社会生态农业CSA、人民经济”等。

    在过去的20多年，温三农发挥巨大舆论影响力，不断推出三农和乡村建设（振兴）理论，试图引导政策制定方向，其基本套路还是2000年左右做杂志社（新闻媒体）那套。

    现在他们并非草根和普通人，而是有参政议政资格的专家，为国家政策层建言献策，掌握重要权利，其影响力范围包括：新闻舆论导向、精神文化、思想理论、高等教育、行业规则和标准。

    有可能，三农专家的某个馊主意被政策层采纳，将导致严重的后果，历史上犯错然后纠错的案例非常多，必须遏制这类涉嫌祸国殃民专家的言论和行为。

    也许以上的判断是错的，欢迎提供相关材料佐证，本着实事求是的原则，得到确证后，我们将承认错误，并修改或增删不合理的部分。

    10多年过去了，为什么还要翻这些陈年旧账？因为我们在成长，需要从过往经历中吸取经验教训，必须追问那些当年没有得到答案的问题，不断总结复盘自我纠错，否则过去踩过的坑都是白白浪费时间精力。

    为了让社会和国家变得更好，不违背信仰和做人原则、实事求是、勇猛精进、不屈不挠。

    此篇目的，不是为了树立对手或敌人，目的是纠错或避免犯错。

    精神解析 #

    从人格上尊重专家，因为其成长过程可能遭受严重的精神创伤，以至于影响后来的行事风格，对此应表示遗憾和谅解，他们是病人，对病人要宽容。

    但不要宽容胡说八道和错误做法，其误导作用可能对社会和他人造成严重负面影响，对事不对人，在事情上要死磕到底，这是唯物主义。

    在奔赴事业的星辰大海和宇宙的征途上，我们要关照人的内心和精神世界，因为人类社会一切好事或坏事几乎都是人造成的，人的行为受精神意识的指导，唯心论也有必要。

    现代精神医学和心理学作为科学体系的历史相对较晚，将精神活动、心理障碍纳入系统化、实证化、可诊断可治疗的医学框架，主要始于19世纪末至20世纪初，弗洛伊德（Sigmund Freud）提出精神分析理论，强调潜意识、童年经验、防御机制等。

    很多人的精神状况没有被鉴定，背后其实是无数沉默的痛苦 —— 他们只是“看起来正常”，却长期在失眠、自我否定、拼命工作中挣扎，对某种错误事物的执着，一定要去证明自己是对的。

    正如心理学家卡尔·荣格所说：“除非意识到你的潜意识，否则潜意识将主导你的生活，而你称之为命运。”

    无论是文革批斗会上的群情激愤、知青上山下乡时的理想炽燃，还是阅兵式上国旗冉冉升起时涌动的爱国豪情……

    或是婚礼现场亲朋相拥的温情洋溢、公司年会颁奖时刻的荣耀激荡，亦或是运动健儿登顶领奖台的胜利狂喜、00后少男少女在演唱会上为偶像迸发的狂热尖叫……

    或是戴锦华教授伫立格瓦拉墓前的潸然落泪，廖晓义老师在舞台上重现革命样板戏的坚毅执着，以及温教授在梁漱溟乡村建设中心举办红卫兵主题生日聚会的特殊癖好……

    —— 这些看似跨度极大、气质迥异的场景，实则共享着同一种深层情感结构。纵使时代语境、价值取向天差地别，其中涌动的强烈情绪与精神原力，在本质上有着相通的内核。

    这种内核以文化模因（meme） 为载体，在集体仪式的场域中实现跨个体、跨场景的传递与发酵。它借助情绪感染的生理唤醒、社会认同的价值锚定、集体亢奋的氛围催化等叠加机制，被快速激活并持续放大，最终凝结为兼具高强度情感体验与共享意义框架的群体性情感，成为维系特定群体联结的精神纽带和信仰，也可称为精神执着。

    而这种在特定场景氛围中被激发的强烈情绪与精神力量，其根源并非孤立的个体体验，而是潜藏于文化基因中的集体性文化传染与符号共鸣。前者依托人际间的情绪传导形成群体共情，后者通过特定文化符号的解码与认同完成价值内化，二者交织构成了这种精神力量的生成逻辑。

    以上用于解析温教授的独特三农和乡村建设方式的精神内核。
