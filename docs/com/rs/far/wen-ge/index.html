    三农和乡村建设的文革愤青

    2013-09-20 Atom.X

    三农和乡村建设组织是培养狭隘的极左民粹的大本营，21世纪文革愤青延续的精神根据地。

    参考相关文章： 戴爷锦华教授讲切.格瓦拉

    激起愤青 #

    温教授有一本书叫《告别百年激进》，核心是反思西方主导的 “激进现代化”路径，纵观三农理论和乡村建设的历史事件，充满灰暗和凄凉，事实证明，他并非温和，反而是非常激进的，极左导向明显。

    我们加入三农和乡建组织，一起在小毛驴农场同吃同住同劳动的经历，体验到最明显的变化是：“我们变得更愤青了”。

    这些组织的特有行为方式，似乎暗示文革尚未过去，也不是在真正搞三农发展和乡村建设，而是在利用不谙世事的学生搞社会运动，似乎有某些特别动机。

    很多人投入1-2年甚至更长时间来做义工，最后却选择远离他们，因为学生们没有成长收获，没有学到什么有益的知识、技能或精神思想，这些组织的导师们起着负面的示范作用，展示很多负面的东西，却没有展示多少优秀的一面。

    “聚是一团火、散是满天灰”，不是温教授说的“散是满天星”，也不是蒲公英飞到各地播种，每次散伙都留下一地鸡毛和狗血，这就是当时我们经历的三农和乡建组织的真实状况。

    说一些具体的经历吧！

    唱红打黑 #

    2012年初刚过春节，小毛驴农场的很多同学都没回来，我们觉得挺奇怪的，为啥农场一下子空了？

    原来他们被乡建团队带到重庆，参加“唱红打黑”活动去了，重庆出钱让学生们去站台。然后不久薄熙来倒台，他们也在媒体上沉寂了一段时间，那时我已南下杭州工作，这中间发生了什么，不得而知。

    温教授喜欢唱红歌，每次在农场聚会，他总是特别兴奋地带头唱，三农乡村建设核心团队有几个人也擅长唱红歌，把人选派到重庆能发挥特长，可以再多开几个乡建基地。

    温教授为代表的那批50年代出生的人，正是年轻气盛的时候，经历过完整的文革和知青上山下乡运动，结果造就了两类人。

    80年代流行伤痕文学，很多知青写下充满精神创伤的回忆录，对那段经历感到厌恶和憎恨，言语总是戴着戾气和不安，因为他们家破人亡或终身严重伤病，留下文革后遗综合症，很多80后出生的人不知道这段历史的细节。

    另一类人很怀念那段激情燃烧的岁月，退休后还经常聚会，一直到老了还充满亢奋，跟随媒体的节奏喊打喊杀，他们不屑“和气生财、韬光养晦”这种说辞，只有斗争和对抗的路可走。

    同在重庆的还有另外一位名师 - 北京地球村环境教育中心的廖晓义，是中国知名的环保活动家和生态文明守护者，我参与过她在巫溪县做的乐和家园，同样也是乡村建设项目。

    廖老师也是50年代生人，同样经历过文革和知青运动，她也喜欢红歌和革命样板戏，比温教授行为更过激的是，她在千人国际会议上激烈批判（大骂）某位嘉宾，面对面地毫不留情地、怒目而视，或当众哭泣大喊大叫（歇斯底里），因为她对相关方感到失望或绝望。

    在西南大学乡村振兴战略研究院的官网上，戴锦华教授和廖晓义老师也被列入特邀研究员。

    我周围有很多生于50年代的长辈和老师，具备以上两种典型特征，上面几位老师大概都属于后者。

    红卫兵 #

    2010.6.12，温教授59岁生日。地点：梁漱溟乡村建设中心，北京市海淀区温泉镇温泉村。

    温三农旗下的乡村建设各团队的上百名学生来给教授庆生，作为小毛驴农场的一员，我也获邀参加。

    走入活动现场大院，到处都是红旗飘飘、大字报、大横幅、学生们穿着绿军装、红五角星帽子、带着红袖标，拿着红缨标枪列队在道路两旁，热烈欢迎温教授和来宾，并给温教授系上红领巾……

    我当时有一种穿越到文革年代身临其境的感觉，温教授就像是伟人毛主席，只有在电影、电视、图片中见到过这样的震撼场景。

    如果有导演想拍革命题材和样板戏，强烈推荐去梁漱溟乡村建设中心，我们一定会帮忙联系沟通，上哪里找这么好的群众演员和主角？

    温三农的乡村建设团队，用实际行动数十年如一日地坚持每天实地演练，才会自然而然达到电影拍摄场景的效果，根本不需要上电影学院和学造型艺术设计，他们天生就会这套功夫，这是天赋。

    曾经在中文搜索引擎baidu上检索“三农、乡村建设、支农、大学生、志愿者、下乡”等关键词，也能看到几乎一样经典的照片，红卫兵的装扮包括：大红色主题、旗帜、帽子、红领巾、红袖章、服装和标识物，这和50年前的红色革命几乎没差别。

    反西方 #

    温教授经常讲“反思现代”和提倡“生态文明”发展，这些理论应该是一种“包容、多元”的精神，他还明确说“让学生们回到农村去，发展一个本地化的内生系统，警惕外来的西方思想和资本主义”。

    在2010-2012年前后，学生们对温三农理论的印象就是警惕西方，旗帜鲜明地反西方的立场，这或许是学生们普遍存在的误解，不知道这个误解时怎么奇妙发生的。

    总之我和学生们都普遍理解温教授的核心思想是：反西方

    我们在不同场合听到温教授发言的不同版本，他一直在反复告诫学生们一些严重问题，反对的清单很长，主要包括：

    反四化：反对 “私有化、市场化、自由化、全球化”

    后来从三农和乡建核心团队的学习会中传递出来的“反对”清单扩展了很多，包括并不限于：

    反八化：反对 “商品化、现代化、工业化、城市化、资本化、国际化/全球化、精英化 …… ”。

    温教授的乡村建设的组织体系自上而下，包括小毛驴的管理层，都透露出这种“反四化、反八化”观点。

    西方宗教 #

    在一次公开讲座或访谈中，温教授关于中西文明差异、宗教一元性与多元性以及其对农业模式影响的论述，这类观点常见于他近年来强调“去西方中心主义”“生态文明”“在地化发展”的演讲中。

    其中提到西方宗教，温教授原话大意如下：

    从事三农工作的年轻朋友的困惑，这属于中国现有主流的教育和思想体系中所贯穿的“西方中心主义思想体系“，在漫长的黑暗的中世纪只许信一个教。因为他们太散乱，谁都管不了谁，而主教是精神权威，国王、贵族认命，只信一个神，是一元的体系。而中国什么神都信，中国是多元化，所以中国适合做生态农业。

    没找到当时讲话的这篇原稿在哪里，知道的朋友请找到网页链接并分享出来。

    连西方宗教也一起反了，还有什么不在反对的清单里，这变成了彻底反西方文明和意识形态，愤青几乎都是反西方的，他不就是愤老么？

    殖民历史 #

    温教授经常喜欢讲：殖民历史、漫长的黑暗的中世纪，在小毛驴的读书清单里，我看过一本《拉丁美洲被切开的血管》。

    他总是替第三世界国家的殖民历史哀叹，这种朴素的情感和大爱无疆是怎么来的？
