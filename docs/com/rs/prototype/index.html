    军事化管理，做任务兑换社区权益

    2025-11-04 Atom.X

    这样的社区对你有没有吸引力？

    没有使用手机的自由、不许摆烂和躺平、不许睡懒觉、定时熄灯、集体吃饭、采取军事化管理。。。

    我随机做过几次面对面的小调查，大多数人的第一反馈都是无法接受的，皱眉摇头、反抗激烈。我随后补充几句解释，他们又都瞪大眼睛说，“这样做就有吸引力了，好期待，真有这样的地方吗？”

    只要听说做任务就有积分并兑换权益，他们立马表示“让苦难（暴风雨）来得更猛烈一些”吧，人们担心的不是苦难本身，也并非忌讳管束和挑战，而是担心没有及时的回报、奖励、补偿。

    我聊过很多人，至今尚未有拒绝该提案的人，也没人宣称该提案不好的，除非他们根本不看或没看懂却胡说八道。对此犹豫不决，只是因为他们质疑自己的意志力和勇气，没有胆量接受挑战。

    那么社区怎么做的问题，就非常明确和简化了。

    参考前文 《社区怎么做？》

    注册身份 #

    注册入住后真实社交，必须向全体社区居民公布：真实姓名、个人简历、性格测试报告；

    经过职业规划师和心理咨询师的鉴定，给每个人贴上职业和性格标签，并且让社区全体居民随时获悉，禁止在现实世界里隐藏身份背景，日常交流可以用化名。

    注册时记录社区成员的多维度数据，还包括：身高体重、BMI指数、精神面貌、侧面和正面拍照。

    阶段性盘点这些宝贵的数据资产，事无巨细地记录。

    目前大多数社区都刮起一种妖风，纷纷采取匿名制，为了保护很多i人的所谓隐私和尊严，你在某个社区可能住了3个月，依然不知道隔壁桌子的那人的真实姓名和职业，你周围有一群人犹如幻影一般存在，你觉得这正常吗？

    宽进严出 #

    宽进：任何人不限年龄和背景都可以进来，只要是合法身份的人，但是要经过注册和背景信息审核，筛查各种身心健康问题，包括严重精神心理疾病，预防自杀或预谋害人者，维护社区安全。

    严出：通过一段时间各种任务的训练，严格的积分筛选后，才能确定是否能最终留下来，成为社区长期居民（超过6个月），大约60%的长期居民，40%的临时居民（不超过6个月，临时性流动人口）。

    作息规律 #

    每天限定使用手机时间为2小时，早上7:00-7:30，中午13:00-14:00，晚上19:00-19:30；

    办公区开门时间8:00-22:00，晚上22:00关灯，除非有特殊任务要求的人，可使用一个24小时开放的专用小办公室；

    多人间宿舍，起床时间6:00，晚上11:00关灯，除非有特殊任务要求的人，可以更改不同作息时间；

    每天有教练给你的任务打卡，一切都是在友善温和的气氛下进行，大家相互之间鼓励和支持，要共同协作才能更好完成任务。

    没有强制，一切都是自愿参与，可以随时退出并退款，退出时按天扣除费用。

    任务打卡 #

    社区布置了许多任务，绝对不能让人闲置或受到冷落，要让人每天不停地动起来，人只要闲下来，就会无聊、惹是生非、干出各种匪夷所思的事，最好每天累得摊在床上，3分钟就能睡着。

    设立社区规则，成员必须完成任务打卡并获得积分，才能延续社区居民资格，积分可兑换社区餐饮住宿游乐就业创业等权益，也就是积分可用于支付住宿和餐饮等各种费用花销。

    合理合法地赚钱，也允许不做任务的人存在，不干涉他们，有人愿意花比原价（更高的价格）在社区短住，或普通游客，近距离围观体验。

    社区不是监狱，不可能长期在这里待腻味了，我们在全国和全世界各地拓展N个合作点，而且社区与城市周围地区频繁互动，绝不可能有生活枯燥乏味的问题。

    对于违反国家法规或社区规则的行为，视问题严重程度给予扣分，分数扣完时清场，结束社区成员资格，需要重新下一轮的申请。

    任务清单 #

    社区成员履行承诺，并很好完成任务，则增加积分，每月做分数评比公报。

    给予优秀者以下奖励：

    • 筛选优秀成员作为实习生和候选人，派驻其他加盟的基地做运营管理；
    • 支持当地就业和创业计划，办理企业注册和税务优惠一条龙服务；
    • 食宿减免，或赠送礼物，给优秀者发钱或生活物资保障；

    唯有奖惩分明，差异对待，才能建立社区秩序和蒸蒸日上的动能。

    根据社区发展情况，将不断优化任务清单，例如：

    • 体育运动

    包括跑步、瑜伽、咏春、射箭、自由格斗、武道等；

    根据社区成员投票，创建一个或多个运动项目的小组，设置组长和监督员，每天做记录打卡，除非有特殊原因，可以免除锻炼计划，若不能履行健身承诺，就会扣分。

    • 公益服务

    包括福利院老人看护、儿童课余活动辅导、垃圾分类和环境卫生、其他类社工和义工活动；

    社区与当地政府、企业、学校、医院事业单位、村民家庭等各种当地社会组织建立事业合作关系，尽可能让社区成员参与当地的公共事务，即使很小的服务项目，例如管理和维护绿化步道、社区健身活动等。

    • 共创项目

    社区管理项目，包括参观接待、餐饮、住宿、媒体、就业创业活动等。

    例行社区会议，全部成员必须参加，讨论公共事务，并分配职责，参与社区营造，例如垃圾分类培训、社区管理、共学计划等等。

    • 训练营

    包括传统文化、国学、非遗（手工艺）、外语、编程、IT、农艺/园艺师等项目。

    成员必须参与社区组织的训练营，或者由成员自发组织的训练营，根据日程表完成训练目标，以拿到考试或职业资格证书为准。

    积分兑换 #

    将根据社区发展情况，动态更新具体的积分兑换（汇率）规则。

    列举任务说明。

    媒体：社区自媒体平台撰稿和运营，每篇稿件获得200积分；
    外语：参加外语训练营，有大量测试，每次测试获得100积分；
    射箭：参加射箭训练营，每天大量训练，击中靶位获得10积分/环；

    例如6人间床位，原房价2000元/人/月，若社区某居民在限定的1个月内完成了几种任务，例如获得跑步500积分，垃圾分类300积分，创业活动1000积分，每个积分等价1元钱，那么该居民节省了1800元，实际只需要支付200元，节省的钱可以返回到社区居民卡里，用于其他方面的消费，也可以直接取现。

    社区创业 #

    如果你住在社区里，有自己的项目可做，但是和社区完全没关系，那么就没有积分。要么通过参与社区里的其他活动，获得积分。

    请主动设计一个游戏规则，把你的项目嫁接到社区，那么你通过在社区做自己的项目，也可以有积分。

    如果你能完成更多的任务，把食宿费用全部减免，还有盈余积分，可能兑现多达数千或数万元每个月，这就成为了一份正式工作。

    欢迎与社区或关联的合作单位签订工作合同，包括全职、兼职、外包等不同性质的工作，我们希望有更多的社区居民能够独立带领团队、承担管理任务，就业创业成功。

    社区成立一个就业创业部门，职责是积极推动就业创业，部门的考核指标就是能促成社区成员参与就业创业的数量和质量，无论是公益或商业项目，尽可能让成员融入当地社区，总之不能让他们闲着。

    初级成员有1周观察期，然后1个月实习期，逐渐晋级为长期住户，通过任务打卡攒下足够积分，最终通过创业项目，在当地落地生根，成为众创社区的支柱。

    军事化管理 #

    因为时代需要，许多人没有精神内驱力、没有目标方向、意志消沉、颓废、堕落。

    网络流传一句话，“现在中国的现状是：死气沉沉的年轻人、生无可恋的中年人、生机勃勃的老年人”

    很多人可能痛恨自己的意志力弱、不够自律、没有自我约束，买了一堆跑鞋和运动器械，但是没有动力按照计划去锻炼，需要一个很强运动氛围的地方，有教练带着一堆人去运动，他/她才有动力跟随。

    这是社区军事化管理运营的根本原因。

    该休息时，好好躺平、反对自由散漫、浪费精力和财物。

    • 于公：为了社会责任，让社区的人进入一种积极的人生状态，并获得实在的成就和进步；

    • 于私：设计全新业态，独树一帜，区别于其他主流社区，精准客户画像，获得流量和效益；

    我们社区里执行日常管理的是职业教官，教官都有非常强的规则意识和执行力。

    教官背景：退役的中国首批航母海军陆战队特种兵，专长多种冷热兵器、驾驶战车坦克、负重奔袭；原省级和国家级传统武术冠军，专长咏春、自由格斗、弓箭等冷兵器。

    这不是凭空捏造和杜撰的网文，我们喜欢与狠人为伍，教官是我们朋友圈真实存在的狠人，我们聚集了一群江浙地区和全国范围最能打的人。

    目的不是让社区的人成为运动员和武术家，而是为了训练意志力，拒绝消极颓废。

    生物界天生就应该竞争，我们支持内卷和竞争，成为卷王，任何事都要做到极致和专业，否则就不要干了。

    因为公众只能记住前三名，甚至只有第一名才能最终生存下来，必须要和业界最顶尖的高手在一起，力争成为某细分领域或行业的第一和之最。

    人可以平凡，但是不能平庸，不能成为庸碌无为之辈。

    严格要求和自律、每日功课、修炼武道、勇猛精进，改变身体素质和心理素质、维持积极的精神状态。

    如果你经常跑马拉松，哪天休息不跑5公里热身，你就觉得浑身不舒服。没有习惯跑步的人，他们跑1公里都觉得很困难。当人们习惯了约束和规则，融入到行为习惯和日常生活工作中，面对常人的困难，他们能轻松自然应对。

    有多大的能力，就要受到多大强度的训练，任何一个行业都是如此，没有白白得来的能力，即使有某种天赋，不训练强化，潜能也会逐渐消退。

    游戏化规则 #

    目前市面的社区都是以“躺平、摆烂”为模型设计的，我们拒绝躺平、摆烂，对摆烂社区不感兴趣。

    很多人头脑简单，缺乏主动性和自驱力，只能被动接受简单的指令和反馈，通过外在动力驱动，所以任何事都要游戏化、好玩、精彩刺激，像中彩票一样，全凭运气。

    我们不想割韭菜，欺负弱者和头脑简单的人，我们天生使命就是服务人，以人为本，让人和社区变得越来越强大，明天更美好。

    因此，我们决定把社区做游戏化规则的设计，优秀的游戏让缺乏自驱力的人成长，不断打怪升级，激发人的潜能，不断自我重生和升级。

    广发英雄帖，欢迎来踢馆，完成挑战，则给予相应的权益和福利，食宿减免，积分奖励机制，参考减肥训练营或各种荒野求生的综艺节目。

    很多人咋一听，觉得规则太多，会导致社区没人敢来，当然规则是违反人的天性，没有任何约束，自由散漫最好，虽然一时很爽，久而久之，这人就变成废物。

    所以人要求进步，终究是要遵守各种规则，才能变成更强的人。

    如果一个社区从一开始就确立了严格的规则制度，筛选出一批长期居民，这种氛围可以带动其他不具备某些良好习惯的人更容易融入。

    优秀的人只占全社会很少一部分比例，由于人口基数很大，所以优秀的人绝对数量还是相当可观的，他们也想找到自己的社区聚拢过去，但是苦于市场上很少有这样的地方。

    我们创造一个，他们就会聚拢过来，成为地表最强的、最顶级的社区。

    没/有出息 #

    在社区体验一段时间后，身心健康状况变化明显，更积极阳光、找到人生方向、能安心学习、也能谈恋爱了、最后工作也有着落。

    看到孩子变化这么大，家长会举着锦旗来感谢社区，几度哽咽落泪，跟记者和旁观吃瓜者说，他们家那个10几年来深陷游戏网瘾的死肥宅儿子，终于脱胎换骨变得出息了。

    游戏视频娱乐公司能够支付大量的税，受到法律保护，若家长要跟大互联网公司斗，几乎都是失败而归，唯有加强自己的内功和免疫力才是取胜之道。

    可能未来几年有数百万不同程度网瘾的人，受此感召，纷纷要求加入社区锻造内功，为了更好抵抗网瘾诱惑陷阱。

    我们有可能创造一种超强的场域，为所有进入社区的人赋能，这是一种非常微妙的能量，可以玄学点说，这是一种精神能量，每个人都有一点点星星之火，汇聚到一起就形成燎原之势。

    我们要的是这种良好的社会效应，并且在每次活动中高声唱流行曲“没出息”，这首歌应该成为社区的区歌，以此激发人变得更有战斗力。

    心法和做法 #

    很多人看到这里，就觉得做事业太难了，上哪里找那么多业界最顶级的人为伍？

    如果你有善于发现的慧眼，你能总结自己的过往经历，几乎每个人都有成为某个领域顶尖人物的潜力，很多人已经做到了，犹如宝藏被土埋得太深，宝藏近在身边咫尺，但是他们自己却浑然不知。

    还有人一看到做到极致和顶级，就马上反驳说，没有钱，不可能做得成。

    如果事情都要靠钱砸出来，那就好办了，能全部用钱解决的事，那能算什么个难题吗？实际在我们经历的互联网大厂，不断有新的内部创业的部门，钱给够，流量、市场、客户和订单都给够，但是这些出生富贵的业务依然9死1生，并不是所有条件都具备，他们就能做出来。

    钱和流量不是第一决定因素，也不是决定项目成败的最重要的因素，最重要和第一因素是人的精神思想到位。

    我和牟其中在一起工作过半年，他说过一句很经典的话“钱是想出来的”，也就是要有足够的想象力，对社会有充分的认知、知识储备丰富、眼界和思路开阔，脑子里有千万条思维线索，纵横交错，把复杂的关系理清楚，当遇到线索被阻断或无法理清的时候，也要很强的精神思维定力，坚持把纠缠不清的部分弄清楚。

    最后都能把一团乱麻整得干净整洁，这堆纤维就具备实用的经济价值，做成丝线和绳子，纺纱织布，再深加工成服装和工具包等高附加值产品。

    因为大量的各种社会和自然资源就在那里闲置着，一个项目成功可能需要100种资源，能把他们串联组合在一起，这个事情就做成了。成功的事业就像一副巨大的拼图，少了其中某一块，拼图就不是完整的，那就没有成功。

    如果一开口就问怎么搞钱和流量，方法错了，这种急功近利的人，大多都守不住钱，也承接不了流量，他们只会挥霍无度，钱和流量成了洪水猛兽，会害他们家破人亡。最好不要碰大钱和流量，老老实实听话上班，还能保平安。

    FAQ #

    当有人咨询关于这个社区的问题时，以下是一些常见问答，Q（提问），A（回答）

    • Q：住在社区里的人，我们方便自己做饭吗？

    A：有以下几种情况，你可以做饭。

    1. 如果你有职业厨师资格证，或预备实习做厨师，那么我们开放后厨的岗位名额，你可以报名；
    2. 如果你有特殊情况，例如生病或有特殊饮食禁忌，不能在公共食堂和餐厅吃饭，我们有一个小厨房，可以申请使用；
    3. 每周一次或每个月一次大型的社区聚餐，大家一起动手做饭，你可以向组织者申报，预定食材并做饭；

    其他情况下，不便自己日常做饭，因为我们都在公共餐饮设施，经济实惠，节省大量时间，因为我们鼓励任何事都职业和专业化发展；

    • Q：听说社区里不让随便用手机？

    A：这是一个误解，我们的社会是自由的，社区内的任何人都能随意用手机。由于很多人加入各种活动组，为了积攒更多任务积分，以至于没有时间经常刷手机。

    不过大部分都很开心，因为他们完成很多社区工作（任务），通过积分兑换住宿和餐饮，为很多待业的同学节省了生活费，还有不少人额外挣到钱，我们也鼓励更多人参与社区任务，直至做到可以签劳动合同的兼职或全职的正式工作。

    因为玩手机少了，更多的时间用于学习、工作和真实的社交，很多人的精神状态变得越来越好，生活质量有整体提升，甚至实现人生大转变。

    与之相反的是，很多人躺平摆烂，经常拿手机刷短剧、打游戏和看娱乐八卦新闻，不单毁了身体，更是毁了脑子。

    那些过度依赖手机的人，除非是职业需要，否则他们只是通过手机打发时间消遣，他们看着周围的人在进步，或许会想到改变，这是社区的一种引力场，我们就是要构建这样的环境。

    • Q: 为什么要做垃圾分类？

    A: 首先有利于环境卫生，没有垃圾分类，夏天尤其恐怖，一天之内的垃圾会发臭，并长出蛆虫。

    食物残余会生出很多苍蝇和蚊虫，也养活很多蟑螂老鼠，食品和各种物品也被老鼠啃咬，冰可能传播疾病。

    更重要的是，垃圾分类属于一种精神文明建设，不会分类的国家参考非洲和印度等落后国家，发达国家干净整洁，都必然有垃圾分类。你可以观察，不会垃圾分类的人和社群，他们也不会整理自己的房间，里里外外都是乱糟糟，他们就缺乏良好的思维逻辑和社会秩序。

    垃圾分类的核心是整理人的内心，也就是把精神思想中的垃圾（坏、劣质）和宝藏（好、优秀）的东西分得清，并能清理得干净整洁。

    如果还有人质疑或反对垃圾分类，那么这种人就应该去非洲和印度生活，他们不属于文明社会。

    对于那些想不明白为啥要垃圾分类的人，不要费太多口舌解释，他们阅历和见识很原始，不清楚更高级的社会如何构建。
