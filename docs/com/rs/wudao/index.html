    穷文富武 #

    商铺供奉关公 #

    关公是个武将，怎么变成了财神爷？ 真正的商人明白，有武力值才能做成生意，这世界是个肉食丛林，缺乏武力保护的商人，必然被吃掉，至少需要武功护体和威慑力。

    武道馆的目标客户和市场定位，应该与商会联合，而非面向普通市民和儿童的业余健身培训班，商+武结合，才能形成良性富足的生态圈。
