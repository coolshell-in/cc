    风险：做了10+网红烂尾项目后，我对涉足互联网行业的反思总结

    资源：依据我的专业背景，可深度链接的人脉关系清单，毫无保留分享

    烂尾 #

    对涉足互联网行业的反思总结

    被过大的流量（洪涝灾害）冲毁淹死

    互联网发展过快、实体行业跟不上

    流量调度过大、产品服务（水利基础设施）能力不足

    落后的实体生产力和“过度夸张”的网络营销能力之间的矛盾! –- 吹牛太过份，实际动手能力做不到。

    人性弱点/缺陷：贪婪、欲望、恐惧、仇恨、傲慢、投机、盗窃、欺诈、腐败、犯罪、违法、堕落……

    颠倒妄想、贪嗔痴慢疑、杀盗淫妄

    软硬件更新迭代、高度工业标准化、复制太快、门槛低、简单

    无法解决基本问题，缺乏业务应用、技术成为闲置工具

    全球名校网课平台- Coursera: Unix操作系统、网络协议、信息安全、数学、AI

    投行智库、投资、策略、培训、发改委

    资料来源：政府网、两会、中央一号文件解读

    同步对标国家战略 —— 一带一路、人类命运共同体及联合国可持续发展目标SDGs，培养兼具全球视野与本土执行力的复合型人才。

    管理体系设计和实施 #

    合伙人分道扬镳？国企合作？老赖欠帐？写商业协议？立规则？风险预案？

    黄勇刚： PSMD（Pivot Structure Modeling and Deploy）管理体系设计和实施

    PSMD培养共同体建模、部署人员，帮助适应分布式自治组织( Distributed Autonomous Community)的需要。它本身是开放、半自助的共同体。

    顶层设计：基于公司法、合伙企业法、民非法等国家法规和行业标准，设计先进的组织体系和治理结构，预防和控制风险，同时狂飙猛进，超越同行竞争者。

    企业管理咨询，失败案例（信息脱敏），国企/私企画像，应对策略

    政府国资入股、干股权、无管理权，攻老板，啃硬骨头，自上而下治理，现代企业管理

    • 0-1 达成共识思想和行动一致

    知识免费：知识付费和培训卖课已经过时

    筛选合作伙伴、创业者老板、投资人，非营利性协会组织、交流互动，提供情绪价值和信任基础。

    带出去：内地和海外游学； 引进来：外来和尚会念经、私享会；

    钱终归都要花出去，不给我们花，就会给别人挥霍。我们帮花钱，至少物有所值，减少损失。
