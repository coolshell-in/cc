    2026-03-10 Atom.X

    行业媒体 #

    做任何一个行业，反正迟早都要做媒体，宜早不宜迟，且最轻量级的创业都是从自媒体开始，请从做媒体切入

    只需要一台电脑和1个人就可以启动事业，每天浏览海量行业资讯，可以在一年内就积累非常丰富的行业和市场情报，并找到各种潜在的机会。

    如果方法得当，通过媒体积累数千个访客（流量），可以销售任何产品和服务，即使不做任何销售，也具备广告价值，目前（2026年3月）以微信、小红书、抖音等平台为例，日均访问3000的页面，若访客来自精准客群（如女性、母婴群体），广告价值约为 60 -300元/天（约 ¥1800 – ¥9000 / 月）。

    如果客户不能亲自来线下实体店，则必须借助自媒体传播，让他们在网络上知道有这样的一个令人惊叹的餐厅的存在，例如。

    • HappyCow

    请尝试在 HappyCow 上给餐厅增加一个坐标点吧，会有附近大都市的老外找到这里，目前扬州没有餐厅入选，距离扬州最近的苏州有几家餐厅，最多餐厅在上海。

    2019年杭州举办VeggieWorld 植物基（素食）展览会，我和HappyCow的创始人—— Eric Brent一起参加了几场活动，并为他做翻译。

    参考文章： 他素食30年，创办了全球最大的素食餐厅搜索网站，只因心中最真实的初衷

    • 力矩中国- 可持续食品新闻

    实时更新国内外新蛋白领域的最新动态，助您发掘植物肉、蛋、奶以及细胞培养肉、蛋、奶的业务拓展、创业和投资机会。除了分享潮流的植物基饮食行业资讯，力矩中国新蛋白基金将持续助力初创企业家和投资者，共同打造一个健康、可持续发展的食品体系。

    还有许多值得推广的国内外行业平台，不在此一一列举。

    关于行业媒体和知识博主，我们周围许多有过交集的资深人士可引荐。

    公益资助/投资 #

    如果我们有与素食和环保相关的独门绝技或特色专长业务，也可以申请大公益慈善基金会的资助或投资，其中包括盖茨基金会和其个人的公益投资。

    这家全球规模最大的公益慈善基金会创始人，推崇与素食环保相关的活动，积极支持植物基饮食和替代蛋白技术，认为其对减缓气候变化、保障粮食安全具有重要意义。

    1. 多次公开倡导植物基饮食的环保价值

      • 在博客（ GatesNotes）、访谈和书籍（如《气候经济与人类未来》）中，盖茨指出：

        “牛肉是温室气体排放的主要来源之一……我们需要创新食品技术来减少环境足迹。”

      • 他强调：发达国家应减少红肉消费，发展中国家则需获得更可持续的蛋白质来源。

    2. 通过个人资金投资替代蛋白企业

      • 盖茨是以下公司的早期投资者（以个人名义，非基金会）：
        • Beyond Meat（植物肉）
        • Impossible Foods（人造肉）
        • Upside Foods（细胞培养肉）
        • Nature’s Fynd（微生物蛋白）

    3. 支持“绿色溢价”理念

      • 在其气候著作中，他提出：清洁技术（包括替代蛋白）目前成本较高，需通过创新降低“绿色溢价”，使其在全球范围内可及。

    我曾经也参加过盖茨基金会中国首席代表李一诺的“诺言”社群活动，为Beyond Meat的市场经理安排餐饮和会展合作。

    国际化 #

    只有到更大的市场，才能实现增长。

    小品牌也可以借助互联网破圈，突破小地方的地域和资源条件限制，到大都市和海外是上策，拓展更宽敞的大道。

    在欧洲吃蔬菜和素食是非常奢侈的，以扬州素馨阁为例，20元/位（约2.5欧元）的价格，在欧洲仅仅能买到一个最简单的小面包。如果要吃一顿自助餐，在德国最廉价的餐厅也要20欧元（约160元/位），素食的价位则更贵。

    中欧的物价高，但很多产品服务质量越来越差，例如德国等老牌发达国家，甚至有明显整体走下坡路的趋势，这背后有难民和新移民政策，熟练的职业技能劳动力不足等问题导致，关于全球排名前10%的发达国家的素食餐厅和市场调研报告，将另创建一个文件陈述，故不在此赘述。

    如果欧美发达国家的食客们知道有这样的一家高质低价的自助餐厅，一定会非常惊叹并蜂拥而至门庭若市。

    但扬州不是国际大都会，很少有老外在此长期居住和旅行，应该到他们的家门口最近的地方开店。

    国际组织 #

    很多国际组织都与素食有关，议题包括动物保护/动物福利、环境保护、卫生健康等，请加入与素食环保相关的慈善/公益基金会、智库，例如：

    • 良食基金

    隶属于中国生物多样性保护与绿色发展基金会（绿发会），致力于推动健康、可持续、公正的食物系统转型。通过“良食峰会”“周一无肉日”“良食对话”等项目，倡导植物优先饮食、减少食物浪费、改善动物福利，融合东方饮食智慧与全球可持续发展目标。

    联系人：良食基金创始人- 简艺

    • 世界农场动物福利协会（CIWF）

    成立于1967年，总部英国，是全球领先的农场动物福利组织。CIWF倡导减少工业化养殖，推广高福利养殖标准，并支持向植物基饮食转型以减轻环境压力和动物痛苦，在全球多国推动政策与企业实践变革。

    联系人：CIWF中国首席代表- 周尊国、食品行业总监- 崔西·琼斯

    • 柳叶刀EAT（EAT–Lancet Commission）

    由国际顶级医学期刊 The Lancet（《柳叶刀》）与非营利组织 EAT基金 于2019年共同发起的一个全球科学倡议，联合发布“行星健康膳食”（Planetary Health Diet），首次提出兼顾人类健康与地球生态的全球科学膳食指南：大幅减少红肉与糖摄入，增加全谷物、豆类、坚果和植物性食物比例。

    联系人：《EAT-柳叶刀》报告的科学转化主任委员、撰稿人- Brent Loken

    • 明绿智库 Brighter Green

    美国纽约智库，聚焦全球食物体系、气候与动物伦理议题。其顾问团队研究畜牧业扩张对发展中国家环境与社会的影响，倡导植物性饮食转型、可持续农业政策，并支持全球南方国家的食品正义与生态农业实践。

    联系人：明绿智库创始人- Mia MacDonald

    联合国分支机构在其官方报告、政策倡议和项目中，明确将减少动物性食品消费、推广可持续膳食（包括植物性饮食）作为应对气候变化、生物多样性丧失和资源压力的重要策略之一，主要包括：

    联合国粮农组织（FAO） #

    ✅ FAO 强烈主张向更可持续的膳食模式转型，反对不可持续的肉类生产与过度消费，其中减少红肉和加工肉类消费是关键倡议。

    主要举措与文件： #

    1. 《2021年世界粮食安全和营养状况》报告

      • 首次系统提出“健康膳食（healthy diets）”，强调植物性食物（全谷物、豆类、水果、蔬菜）应占主导，动物性食品需适量。
      • 指出当前全球食品系统导致30%以上温室气体排放，畜牧业是主要来源之一。

    2. EAT–FAO 可持续膳食指南合作

      • 与非营利组织EAT合作开发《可持续膳食指南制定框架》，供各国政府参考。
      • 明确建议：高收入国家应显著减少红肉摄入，转向植物蛋白。

    3. “手拉手”行动计划（Hand-in-Hand Initiative）

      • 利用地理空间数据识别农业与环境热点，推动低碳农业和可持续消费。
      • 支持替代蛋白（如昆虫蛋白、植物基蛋白）在发展中国家的应用研究。

    4. 全球重要农业文化遗产系统（GIAHS）

      • 保护传统农耕系统（如中国的稻鱼鸭共生），其中许多系统天然依赖植物为主、动物为辅的循环模式。

    世界卫生组织（WHO） #

    WHO 的重点是健康驱动，承认并支持其与环境目标的协同。从公共健康角度出发，将减少红肉和加工肉类摄入列为预防慢性病的关键措施，并承认其环境协同效益。

    主要举措与文件： #

    1. 2015年国际癌症研究机构（IARC）

      • 将加工肉类列为1类致癌物（与烟草同级），红肉列为2A类（可能致癌）。
      • 该结论被环保组织广泛引用以支持减少肉类消费。

    2. 《健康星球，健康人类》联合行动计划（与UNESCO等合作）

      • 明确指出：“不健康的饮食模式（高肉、高加工食品）”。
      • 倡导“一体健康（One Health）”理念，将人类健康、动物健康与生态系统健康统筹考虑。

    3. 支持国家制定膳食指南纳入可持续性原则

      • 鼓励各国（如瑞典、巴西）在官方膳食指南中加入环境影响评估，推荐植物性饮食。

    联合国环境规划署（UNEP） #

    UNEP UNEP 是最明确将“减少肉类消费”与环境保护直接挂钩的联合国机构，最接近“素食环保”倡导，表述为“可持续膳食转型”。

    主要举措与文件： #

    1. 《2010年排放差距报告》及后续年度报告

      • 多次指出：畜牧业占全球人为温室气体排放的14.5%以上（后修正为约12%），是减排重点领域。

    2. 《全球资源展望》（Global Resources Outlook）

      • 强调：转向植物性饮食可减少70%以上的农业用地需求和50%以上的水资源消耗。

    3. “可持续生活方式”倡议（Sustainable Lifestyles）

      • 在“吃什么（What We Eat）”板块中，明确建议：

        “减少肉类和乳制品消费，增加豆类、坚果、全谷物和蔬菜摄入，是个人减碳最有效的方式之一。”

    4. 支持“世界无肉日”“国际素食日”等民间活动的科学背书

      • 多次在社交媒体和报告中引用相关数据，肯定其环保意义。

    5. 牵头“联合国生态系统恢复十年”（2021–2030）

      • 指出：减少牧场扩张是恢复退化土地的关键，而改变膳食结构可缓解土地压力。

    🌱 共同趋势：三大机构在科学报告、政策建议和全球倡议中，一致认为——向以植物为主的可持续膳食转型，是实现气候目标、生物多样性保护和公共健康的关键路径之一。

    民间外交 #

    本人曾多次参与以上多个国际组织与“素食、健康、食品安全”相关议题的会议和活动，并为其提供IT技术支持和翻译服务。

    与会者因此能深刻体会和理解，很多发达国家的精英阶层和意见领袖们很推崇素食、环保、慈善/公益，并热衷在各种顶级的峰会上讨论。无论他们是否违心在舞台上表演或为了博得一个好名声，至少讨论和传播这些话题是非常合适的，除此之外，还有什么更好的话题可讨论的吗？

    因为这些话题符合全人类命运共同体和联合国可持续发展目标SDGs中定义的全人类共同利益，否则为了各自利益，无法相互对话，势必导致更多的纷争和冲突。

    后来我逐渐明白，我们服务的这些大公益基金会和国际组织，其实都承担了民间外交的职能。瞬间让自我身份认同感得到极大的升华，所以我们一个看似不起眼的餐厅服务工作，很可能会撬动国际关系的改善。

    因此厨师和服务员也可以是高大上的职业，人的成功和荣誉感来自于对人类整体的利益贡献，把微弱个体的工作与全体人类命运绑定在一起。

    因党派分歧谈判破裂，处于一种无法沟通对话，官方人物放不下脸面，而又渴望突破的状况下，往往会通过民间外交途径化解分歧，有许多历史案例。

    二战期间国共生死较量，面临外敌入侵国家危亡之际，主要通过民主党派从中斡旋，促成国共合作统一抗战，其中一些关键人物是民国时期念佛的居士，包括：欧阳竟无、杨文会、吕澂、赵朴初、梁漱溟。

    居士们吃素不杀生，作为中间人，形象更友好，善于化解分歧。

    当两国的官方外交因利益冲突，谈判受阻时，也是由民间外交建立关系，通过非政府人士、文化团体、体育代表、学者、媒体或友好组织等“中间人”传话来沟通，进行间接接触、传递信号、试探立场、缓和气氛，为正式谈判创造条件。

    中美关系、中日关系都是如此，通常的套路都是互派体育、文艺、旅游团体交流，也就是官方在台面上下不来，但是不耽误底下这帮人互访，坐在一起吃喝娱乐，吃喝是最简单的外交和社交方式，全球通用。

    关键方法 #

    社会方方面面的马太效应都很明显：“强者愈强、弱者愈弱、赢家通吃”。

    从小区域和某个细分行业领域开始成为第一名，然后逐步扩展到更大和更远的区域，并成为第一名。

    很多时候只有成为第一名才能活下来，只有第一名才能被人记住，并获得足够的生存发展的资源，甚至第2-3名都没人记得住其名号，排名更靠后的要么终生平庸，要么逐渐消亡。

    其实要活下来的方法也很简单，需要能坚持和忍耐就可以，因为无法适应大环境的变化，可能大多数店家都要倒闭关门。

    每个人都有自己的特长，几乎每个人都有成为第一名的潜质，为了生存，一定要努力成为某个细分领域第一名。这需要认识和理解自己，发掘自身的特长兴趣，并发展为高超的职业技能。

    成为第一名的起步阶段并不需要高超的技巧，大多数也毋需高智商，只要老实本分地下笨功夫就可以，于是熟能生巧，达到技艺精湛和出神入化的境界，需要非常专注、心无旁骛、没有杂念，很多事业有成的高技能专业人士都达到这种心境，其修炼过程用佛法经典的词叫“勇猛精进、禅定”。

    高超的职业技能和事业成功，带来丰厚的物质回报，以及精神上的荣誉感、成就感和满足，因此职业技能和事业也是一个非常好的修炼法门，门槛相对比较低。因此鄙人一向不屑于好高骛远的人，缺乏有竞争力的职业技能，连基本生存都保障不了，却空谈什么精神追求，即使有些所谓思想大多也是妄想。

    老实本分就是遵守法规，因为很多人玩花招投机取巧，总想走捷径，最后都做成了打法律擦边球和灰黑色产业链（灰黑产），他们迟早都要玩完，即使没有官方制裁，缺乏稳固忠实的客户，他们也活不久。

    希望通过非正当劳动、非常规经营或非合法稳定渠道获得意外之财。其核心特征是：来得突然、数额较大、来源偶然，且常伴随不确定性甚至道德或法律风险，“一夜暴富、人无横财不富”等一些俗话形容这种心态，这在社会流传甚广的俗话，给公众带来非常负面的影响，所以需要将“正念”刻进企业和组织的文化基因中，并且每天不断强化，像军队和运动员一样强化训练才不容易退转，因为人非常容易堕落，进化论应受到质疑，退化论更应成为科学。

    这是鄙人给数十家企业管理团队做过咨询顾问后得出的经验总结，失败教训的成本非常高昂，成功的关键方法或失败的缘由，亘古至今却一直没有变，这让人颇为费解，为啥不吸取教训呢？关键原因可能是人顽固的妄念太多干扰所致。

    价值观 #

    现代企业和公益组织有企业文化或组织文化（愿景、使命、价值观），也可简称为价值观，价值观是组织的精神内核，一切结果都受价值观的牵引。

    所以我们有经验的人事部门主管招聘候选人的时候主要看2点：价值观和职业技能。

    遵守法规仅能让事业及格，好的价值观能让事业变得优秀和出类拔萃，所以要做得超越法规的要求，从事业创始之初就不断打磨价值观。

    价值观不需要我们自作聪明地再去发明创造，因为历史上有无数的先进模范案例，直接抄古圣先贤的名言警句作教材，照着做就好了，非常简单吧。

    学习方法 #

    按照国际通行的版权著作权法，超过70年的著作，很多都没有版权，任何人都可以直接拿去用，因此各种流通千年的古典名著变得廉价，或完全免费，甚至你去学习，还要倒贴钱或补贴交通食宿给你，例如佛教、基督教和伊斯兰等宗教团体的名门正派都是如此。

    所以善于学习的同学，一定会找到免费的学习资料和好导师，收费太高的大多是邪门歪道，几万块的培训课，大多也是割韭菜的。

    记住一条地球普遍法则，鄙人不敢妄言是宇宙普遍法则，很多好东西确实是免费或廉价的，甚至让人感觉不到其存在而不会珍惜，犹如太阳光、水、空气，因为好东西的创造者都希望我们变得优秀，而优秀的继承人也会投入毕生的精力和财物去支持好东西的传播推广，所以我们总是能得到免费或廉价的好东西。

    引用佛法，好导师是善知识、价值观是菩提心、好教材是法宝。
