    概述做产品设计的框架

    2026-05-17 Atom.X

    请回答以下问题，并尽可能详细！

    解决什么问题？ #

    题目说明：用客户自己的语言描述这个问题，比如他平时怎么跟朋友抱怨这件事的？不是你分析的，是他说的原话。 示例：「我每天都在忙，但一到年底绩效会议我说不出自己做了什么」

    希望拓展线上营收，但还没有IT和网络营销团队，不知道怎么做。

    凭什么是你？ #

    题目说明：从三问中至少答一个：①你自己经历过这个困境并且解决了吗？②你帮别人解决过，有具体的结果吗？③你有让你做这件事成本更低的资源或优势吗？答「是」越多，产品说服力越强。

    1. 我已经服务了数十个中小企业、地方政府、投资者，为企业数字化转型、电子商务和互联网营销、以及团队建设做咨询培训；

    2. 在10年内我做了超过10个网红项目，成功推出业界顶流的产品和服务，因此积累了个人丰富的行业经验和技能；

    3. 我在数家业界头部的互联网大厂的运营管理部门工作过，依然有很多的同事、合作伙伴、老朋友在排名前几位的行业大厂里工作，可调用最顶级的技术研发和运营管理专家资源网络；

    产品名称 #

    按公式填写你的产品名：【对象】+【时间或数量】+【核心结果】+【产品形式】 示例：全职宝妈·90天小红书从0粉到变现陪跑计划

    传统企业3个月从0开始数字化转型和团队建设计划

    产品卖点 #

    按公式填写你产品的一句话卖点：帮【具体的谁】，在【多长时间内】，做到【具体的结果】。一句话，不超过60字。 示例：帮带娃在家的全职宝妈，90天在小红书积累1000个精准粉丝，拿到第一笔广告或带货收入

    帮传统企业，3个月内实现数字化转型或升级，招募并培训团队，提升工作效率

    产品里有什么？ #

    先想清楚你这个产品最主要帮用户拿到什么结果？然后往回推：要拿到这个结果，用户需要什么？只写跟核心结果直接相关的内容，3-5条。 示例：​ ① 21天不节食减5斤的饮食方案（按上班时间定制）​ ② 配合碎片时间的运动计划​ ③ 每天打卡群监督执行​ ④ 随时答疑（遇到问题当天回复）

    –

    以下内容适用于一般工业级和民用级产品的企业，为了更精准服务，列举应用案例主要为农文旅行业。

    1. 基础常识：个体和企业在剧烈变革的信息化时代的风险和机遇；
    2. 合规审查：公司法、规章制度、互联网法规、财税信息管理等；
    3. 高效工具：共享文档、客户关系管理CRM应用、网络信息安全；
    4. 人工智能：智能体辅助客服、营销、财务和数据分析的自动化；
    5. 品类创新：高性价比的选品、标准认证、产品技术和服务创新；
    6. 营销创意：品牌设计、媒体传播、市场运营（用户/产品/平台）；
    7. 团队建设：IT和AI时代的人才招募、企业文化、培训体系锻造；

    如何定价 #

    简单说说你是怎么推出这个价格的？推算过程不需要很长，把你用上面那个参照得出价格的过程说清楚就行。 示例：我搜了市面上类似的职场晋升课，低端99-299，中端599-1299。我有3个学员案例但粉丝只有2000，信任基础不够站高端，定699站中端。

    此价格参考我们2014年开始为数十家农文旅行业的咨询培训、以及为投资调研机构服务的案例，当时服务价格为 2000元/小时，考虑到2019年之后的经济环境变化，且批量为多家企业同时服务，故将价格定为2000元/3个月，平均算下来每个月才600多元，已经是地板价了。

    认真描述产品故事·第一段： #

    一个具体的年份或事件，说出你当时卡住的那个具体状态（不只是感觉「压力很大」，是「每天打开表格，不知道从哪一件开始」这种） 一个具体的低谷细节。要有时间、有场景、有卡住的状态。不需要你多惨，需要用户读后能感觉到「说的就是我」

    每天打开手机，微信里各种行业资讯看的太多，对行业各种宏观趋势和微观的技术细节似乎都了如指掌，但实际互联网上的内容纷繁复杂、机会或陷阱真假难辨，怎么把好技术和平台用到企业的产品和服务改进中呢？

    每天下班躺着，就想刷手机短视频，或打打游戏停不下来，越晚就越睡不着，第二天上班脑袋昏沉沉，这种日子慢慢把人熬成了缺乏人生动力的咸鱼。。。

    认真描述产品故事·第二段： #

    我遇到了一个转折点。说明：一件具体发生的事，让你开始改变。越具体越好！是哪一天，发生了什么，你的第一反应是什么？

    当时我无法直接回答客户以上的诸多问题，直到2014年某天，我发现Coursera等全球名校和顶尖科技企业的公开课平台，找到大量新兴互联网技术教程的源头，感觉国产的那些被人翻译复制和篡改后的课程简直就不忍直视，这也就解释了大多数中小公司的团队所接受的培训质量（较差）与其项目产品质量（较差）是成正比的关系。

    认真描述产品故事·第三段： #

    我找到了一个东西。说明：这个东西是什么，它怎么帮了你。 如果是方法，强调「是方法，不是天赋」这意味着用户也可以学。

    通常一手的资源都是来自于海外，类似Coursera等网课平台和科技公司的开源社区里，有前沿技术动向和应用方法的细节。例如 名企 Google IT和Facebook数字营销，以及名校斯坦福的数学、人工智能和深度学习等课程。这些不仅仅是课程，而且附带了云计算环境下的真实训练系统。

    认真描述产品故事·第四段： #

    我用它拿到了结果。说明：具体的、可感知的结果。不是「进步了很多」，是一个有数字、有时间、有可以被验证的事实。 需要这个结果，用户能不能在脑子里看到？能不能被核实？

    通过考试后的证书，对于跨境电商企业招聘选人有效，其技能可以直接用于跨境电商运营管理和营销实操，尤其适合于IT技术和计算机编程等相关专业学习训练。

    认真描述产品故事·第五段： #

    我把它分享给你了。说明：产品出场。不是在推销，是「我只是个传递者」的感觉。我只是把这个东西带到了更多人面前。

    很多世界顶级公司和名校推出的课程、IT工具和应用平台都免费，且大量的开发者社区里的开源项目公布在互联网上，或只需要非常低的学习成本，让每个人都能负担得起，为什么不用最好的教程和生产力工具呢？

    而我只是一个陪练，当有人需要时，就及时支援或给予指导。

    书写产品亮点·画面句① #

    检验标准：读者能不能在脑子里看到一个具体的场景？能看到就留，看不到就改。 不要写：「效率提升了」「状态好多了」（这是评价句）​ 要写：「那周你五点半下班，还有时间去接孩子」（这是画面句）

    用国际平台和工具应用，在英文工作环境中训练，告别了传统公司的土鳖工作方式，每晚都听着英文播客入眠，大量地输入和输出，连英语水平都在不自觉地提高，3个月克服了过去10多年学的聋哑英语。

    书写产品亮点·画面句② #

    根据教练布置的作业任务，每晚都要刷2节课并测试，每周至少3次健身打卡，下班再也没时间无聊地刷短剧和打游戏了，每天都过得很充实。

    书写产品亮点·画面句③ #

    在公司里率先应用了智能体部署，相比其他传统业务组，我的工作量和效率都得到2倍提升，而且实际工作没那么累，上班大多数时间其实都在研究新技术，在内部路演时能挺着腰杆大声讲解前沿业务模式和技术应用，人事领导暗示我今年肯定晋升加薪。而曾经在会议上，我只是一个坐在最后一排，默默听其他人发言的小白。

    提前准备好回复质疑①： #

    「你能做到，为什么代表我也能做到？」 破解方式：不只说你自己的故事，说你帮过的人的结果。别人的结果，才能回答「我也可以吗」这个问题。

    其实大多数人的智商差多，差异大的是意志力、精神激励，这正是我们作为教练（过来人）的作用

    提前准备好回复质疑②： #

    「这是真的吗？有没有可能是编的？」破解方式：用一个可以被核实的细节，不用形容词。有时间、有平台、有数字！用户可以去查。

    我在国内学历只有大专，但是我靠自学能力，有机会去头部互联网大厂和10多个业界知名项目中锻造过。

    当然，与学历、智商、家庭背景、经济状况无关，我可以，你也可以，我经历的详情可查阅个人网址 https://atomx.cc/zh/

    提前准备好回复质疑③： #

    「凭什么是你？别人也有类似的产品。」破解方式：说清楚你在一件具体的事上和别人不一样在哪里。不说「我更专业」，说「市面上大多数XX课教你YY，我这个只解决一件事：ZZ」

    市面上大多数互联网相关的培训课教单一技能，我的方案解决复合能力问题 – 人的学习能力和成长发展

    你还有什么想补充的不？没有限制 #

    可以补充你觉得这份考卷没有问到、但对你的产品很重要的事情。比如：你已经有的真实客户反馈、你觉得能让评委更想买你这个产品的细节、你现在面临的具体困难、或者其他你觉得我们应该知道的。​

    ​“世上无难事、只怕有心人”，也就是只要足够上心，训练专注力，心无杂念，没有什么学不会和掌握不了的高超技能，很多事都是下笨功夫，认真老实地打磨基本功，不要急功近利，没有捷径可走。

    现实的世界中，很多人数字中毒、信息过载，迷失了人生方向，分不清机会和陷阱，在无聊混沌中消遣大好时光，浪费无数的免费优质资源，花重金去追求虚华胡哨毫无营养的东西。
