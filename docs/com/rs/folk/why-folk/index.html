    为什么要做民俗活动？

    2025-10-30 Atom.X

    论公共空间的社会功能和价值。

    公共空间 #

    以书房为例，真正的书房是一个专用于看书学习的空间，没有床铺，不能吃喝，不能用于其他用途。

    一般普通人家里没有书房，因为中国房子太贵，舍不得专用一间房专业看书写字，只有少数真正的读书人家庭才会设置书房。

    最重要的原因，现代人都把注意力放在电子屏幕上，手机随时随地看资讯，所以也觉得没必要看书，更没必要设置一个专用于学习和存放书的书房，看纸质书实在非常昂贵耗费成本和时间，还不如手机和电子资讯来得方便。

    但几乎人人都向往学霸、崇拜知识、附庸风雅，即使不热爱学习，也会经常想去蹭一下热爱学习的氛围。

    富豪家里房子多，即使不看书，在经济条件充裕的情况下，一般也会设置一个专用书房，把文房四宝和各种经史子集摆上。

    根据市场情况，既然全民都有需求，于是公共空间的书房 - 规模更大的图书馆诞生了。

    其他类似的公共空间还包括健身房、体育馆等等。很多年轻朋友乐于去健身房打卡，在朋友圈拍照，亮健身装备，在健身房更容易找到帅哥美女聊天，真正费力流汗的运动是不会的。

    尽管这些空间里过去很多年，也只产生了少数几个运动健将，比赛并获得前几名的成绩更是少数人，不可能所有人都拿很高的分数，即使经常使用这些空间的大多数人一直都是平庸的，但公共空间依然是非常有必要性的。

    红花需要绿叶和青草地衬托，体育明星和学霸也需要普通人围观当灯泡，明星和学霸被围在中间才显得更璀璨夺目，并激发更多人参与进去，才可能产生更多的明星和学霸，然后这些少数精英就改变了整个时代。

    所以公共空间的必要性，就在于低成本地提供更好的公共服务，让全社会变得更强更好。

    中国建造和管理的公共空间，功能很实用且非常廉价甚至完全免费，一般都是亏钱的，不得不让人感叹领导决策者之英明，公共空间体现了国家制度优越性，是执政者建设社会主义的初心承诺，所以中国基建都做得很大很好很先进，当之无愧世界第一。

    官办的公共空间数量较少，无法完全满足全社会的需求，于是诞生了很多私人开办的（商业化）空间，经营面积和客户规模的体量一般都不大，但开在你家社区楼下，真滴很方便，老板可能也很赚钱。

    如果去健身房的人都很勤奋，可能老板就不怎么挣钱，甚至要亏钱了，其实更多挣的是那些办卡却不去的人，所以，精明的老板绝不会看不起那些体弱的人。当然那些很强悍的人在健身房也是一道靓丽的风景，他们本身就具备自动吸引力，主动帮老板招揽生意，所以无论强人弱者都是受欢迎的客户，只要好好办会员卡，都要笑脸相迎以诚相待。

    儿童区 #

    在公共空间专业配置儿童活动区，将极大地解放家长的双手，提供的这一增值服务，有利于带儿童的家庭融入，因此产生更多附加值，包括商业消费和情感价值，如果孩子喜欢某个地方，家长也会应孩子的要求经常光顾那里。

    儿童区的配置，需要参考一些先进案例。

    民俗活动 #

    书房太冷静高雅，民众爱热闹市井，于是要出门参加集体活动

    很多展览陈列馆和博物馆的活动是死的，而不是活的，人们需要参加活动。

    活动 - 人们是活的，并促发更多人参与，一起行动起来。

    什么样的活动能调动区域内所有人都参与，且参与者不分社会身份地位和年龄段？

    民俗活动适合全民参与，最热闹的场景，例如市集、庙会、节日庆典等，在全世界都是如此，不同地区的民俗风情习惯不同，对这类活动的叫法不一样，其本质都是一样的。

    民俗集会能够装下所有类型的活动。

    仿古装 #

    仿古服装cosplay，仅节日活动穿一下，有一些新式设计的汉唐风古装，很生活化，适合日常穿着，且价格特别实惠。

    例如电商平台搜索：汉唐风衣

    体育竞技游乐场 #

    参考综艺节目： 跑男 - 奔跑吧兄弟

    现代人心理压力太大，已经受到太多挫折，不能让人总是遭受失败和打击，因此竞技性的活动难度不能太高，需要设计更容易参与的低门槛活动，有一点成就感和荣誉感，娱乐为主。

    游乐场设施包括翻墙、爬绳梯、走平衡木、跳过障碍物等，设计参考如下：

    • 拓展训练场 / 户外拓展设施（Outdoor Challenge Course / Adventure Course）
    • 体能训练综合体（Fitness Trail / Parcourse）
    • 军事训练场（Military Training Ground / Assault Course）
    • 障碍训练场 / 障碍课程（Obstacle Course）/儿童障碍乐园 /趣味挑战赛道

    参加比赛要拿奖，获得一个“少侠”的勋章，有机会上媒体，得到各种关注，为了帅、靓。

    民俗演出 #

    人们只能记住前三名，村超已经在广西、贵州、江苏遍地开花，后面跟进的省份还去搞篮球和足球项目，就没有机会再获得全国关注的流量，因此一定要开发具备本土民俗特色的项目。

    目前能让全国人民都知道的传统体育民俗项目，应该算是龙舟，尤其广东的赛龙舟。

    广东佛山：叠滘龙船，开飙！

    作为传统项目，舞龙舞狮依然有市场，但也更需要功夫创新，以射箭项目为例。

    射箭+功夫 #

    临安有钱王陵园和钱王村，杭州人都知道“钱王射潮”的典故，但没有与此相关传承的射箭活动，值得文创团队去好好考古和设计。

    能发动一个地区的所有人都参与才算成功，参考村超的接地气，竞赛奖品是本地土特产：猪、羊、鸡鸭鹅、鱼、水果、大米。

    咏春和射箭这类实战格斗项目，很多人觉得这些动作大多都不够帅，甚至看起来很呆板，但外行看热闹而不会看门道，因此有实战力的老师傅都难以招生。

    其实好看的武术动作，就是街头杂耍和卖艺，也叫花拳绣腿，各种花哨动作是普通人喜闻乐见的，帅靓的动作包括：旋风腿、空翻、把人打得飞起来。

    现状迫使真功夫要创新，包括动作、服装、道具等全套的行头。

    • 动作：多人组合的箭阵，将跑酷动作加进去，前后空翻+跪地射箭。

    • 服装：穿上中西古今合璧实用的行头，类似漫威英雄联盟、日本忍者、或明朝锦衣卫的复活，带上护膝、护肘、护腕、护肩的甲胄，看起来与现代科幻片都毫无违和感。

    • 道具：气球飘在天上，挂着红色大字的条幅标语，两个武士交叉向对方侧上空射击，气球啪地一声破裂，喷出金色万花筒里的飘带，伴随鼓乐齐鸣，现场气氛达到热烈顶点，这是最新式的剪彩仪式。

    组建一支武士团队，专职培训和表演，因此武术生活化，日常天天操练，而非偶尔演艺一次。

    参考：台湾 九天民俗技艺团，玩+练+赛+演，综合性的专业功夫演艺团体。九天玄女战鼓，传授击鼓方法，表演性质，敲锣打鼓特别能震撼人心和吸引人气。

    武术服装 #

    服装很能说明当前整个社会面的问题，功能实用+外观好看+文化内涵，极少案例这三点同时都能做到，而且很多武术服装简直就是个笑话，譬如：

    • 义和团
      很多欧洲人慕名到中国学武术，穿着那一套花花绿绿的衣服，拿着各种刀枪棍棒，站在一起拍照，看起来像是西游记里成群结队的妖魔鬼怪，或更接近最经典的（举着大刀摆出手势）一群义和团摆拍照。

    当年义和团就是要打洋人的，直至一百年后，洋人纷纷来拜师学武术，也穿上义和团的花衣服，认祖归宗，这也算是文化上的成功吧。

    • 江湖门派
      各大门派都有特别容易识别的独特服装，例如武当派和少林派，长发和秃头，道袍和僧服，特征分别很明显，几乎所有人都认识。

    其他小门派就没有服装上的独特性，太极也算，但是大部分太极服太丑了，像是一坨睡衣。

    为何这些传统武林门派的衣服都特别宽大？因为可以遮住身体，让人看不到身体动作线条，而发源于西方的现代体育运动项目，都是穿着紧身衣，甚至接近裸体，一定要把人体每个轮廓线条都呈现出来，东西方简直就是两个极端。

    某次在欧洲的街上，我和一个教会的老朋友走路，旁边有一个女生骑自行车过去，穿着那种非常夸张的贴身运动裤，几乎臀部线条都露出来，这个老太太叫了一声上帝，说“人体很是太美，但是让人看了简直就像犯罪”。

    两个极端都是有问题的，所以需要一种折中的设计方案，关于更好的不分门派的传统武术服装，目前我们还没有在市场上看到，这个市场也是非常大的。

    关于武术服装，参考即将发布的文章 《武术服装史考据》。

    文化不自信 #

    我们在文化上在过去很多年是非常不自信的，有很多方面的体现，例如：生日歌、生日蛋糕、白色婚礼服、诺贝尔、马拉松等。

    有那么多优秀的传统体育竞技活动，随便挑选几个出来都能做得非常隆重好玩，但是为何最有名的还是马拉松和篮球足球？

    预计在未来10-20年，中国会出现新的名号，替代马拉松的在体育竞赛方面的最亮名号，替代诺贝尔在科学奖项上的最亮名号，因为中国实在太能拼搏了，一定有成果产出。

    生日歌 #

    如果谁能首创中文的生日歌，必定会引起全民的注意力，中文版生日歌诞生那天，那就是真正的文化自信的重要象征。

    发动各大音乐平台、古诗词、曲艺文化社团，一定能选出适合当地人的生日歌，根据区域划分，因为民族信仰语言风俗的不同，至少每种大的区域方言都有一首生日歌。

    民俗活动得到官方的支持，就更容易推广。

    集会烧火🔥 #

    为什么全世界的公众集会活动都要烧火？冬天可以理解，火让人感到温暖，所以容易聚集人群，但是酷暑盛夏，也有很多人喜欢篝火晚会，为何大家都不约而同喜欢聚在火堆前？

    只要烧一把火，很快就能聚集人气，很多词可以形容，例如：红红火火、热情似火、火热赤忱，参考前文 《烧一把火，人心自聚 - 论火的“社交引力”》

    若善于把火玩好，这活动也必定办得红红火火，热闹非凡！

    更玄学一点的说法，火是沟通天地和神明的信号，这就是我们看到全世界无论任何一个宗派信仰，从西方的基督教，到东方的道教、佛教，都要摆祭坛，点上火。

    所谓人间烟火气，一种吸引人聚集和温暖人心的力量，大约最底层的原理来自于神明，通过火赐予人类的能量。

    家庭参与 #

    只做中式民俗文化

    族谱、家训/家规、庆生、纪念日、家族合照

    家族合照标准：祠堂，统一服装风格，轻微化妆，裱框。

    现有的家族合照缺陷：毛线衣、棉袄、大衣、学生服，各种颜色和款型，排列错乱，风格不统一。

    基于族谱考古发掘，可以开发很多有意思的玩法，例如家族社交圈。

    查一下族谱，我们徐氏家族，最早是否从江苏泗洪出来的？约3000年前徐州那一带，我家是东海堂，通过网络能找到很多外地从未谋面的上几代的亲戚们。

    拍摄合作 #

    一条、二更

    找专业的合作方拍摄，民俗类重大节庆活动，例如九天民俗技艺团等。

    民俗出版物 #

    《汉声杂志》

    这是全球最早向世界传播中国民间民俗风物的杂志，也是我见过最好的民俗出版物，《舌尖上的中国》一季和很多央视的纪录片的策划和田野调查都是找他们做的，汉声编辑部还承担了很多中国官方的民俗文化工程。

    从汉声衍生了全国各地的很多项目，例如宁波“慈城古镇的天工之城”，很遗憾这个与地方国企地产商的合作失败了。

    10年前我在宁波慈城古镇一场失败的特色小镇实验中，我犯了那些错误？

    忻苹 - DIY产业的研究与实践者，通过她的案例，可以了解很多地方做非遗手工DIY项目的陷阱，其实本质是合作方的人性缺陷，第一步识别人最重要。
