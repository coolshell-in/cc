    《佛说八大人觉经》

    2026-01-03 Atom.X

    《佛说八大人觉经》，简称《八大人觉经》，是佛教一部极为精要的早期汉译经典。全文仅三百余字，却义理深邃、警策有力，历来被奉为修行者日诵常省的重要典籍。此经与《四十二章经》《佛遗教经》并称为“佛遗三经”，在中国佛教传统中具有广泛影响。

    相传本经由东汉安世高所译（学界对此尚有争议），虽篇幅短小，但融合了小乘的出离精神与大乘的慈悲愿行，既强调个人修持，又导向普度众生，是一条由自利走向利他的完整修行路径。

    经中提出“八大人觉”——即八种觉悟，乃“大人”（指发菩提心、行菩萨道的修行者）所应恒常思惟、践行的正见与准则。其内容层层递进：从观照世间无常、远离贪欲懈怠，到勤修智慧、广行布施，最终发起大乘菩提心，愿代众生受苦，令其得究竟安乐。

    以下为《佛说八大人觉经》全文（依据《大正藏》第十七册通行本）：

    佛说八大人觉经 #

    后汉安世高译

    为佛弟子，常于昼夜，至心诵念，八大人觉：

    第一觉悟：世间无常，国土危脆；四大苦空，五阴无我；生灭变异，虚伪无主；心是恶源，形为罪薮。如是观察，渐离生死。

    第二觉知：多欲为苦，生死疲劳，从贪欲起；少欲无为，身心自在。

    第三觉知：心无厌足，惟得多求，增长罪恶；菩萨不尔，常念知足，安贫守道，惟慧是业。

    第四觉知：懈怠堕落；常行精进，破烦恼恶，摧伏四魔，出阴界狱。

    第五觉知：愚痴生死；菩萨常念，广学多闻，增长智慧，成就辩才，教化一切，悉以大乐。

    第六觉知：贫苦多怨，横结恶缘；菩萨布施，等念怨亲，不念旧恶，不憎恶人。

    第七觉知：五欲过患；虽为俗人，不染世乐。常念三衣、瓦钵、法器，志愿出家，守道清白，梵行高远，慈悲一切。

    第八觉知：生死炽然，苦恼无量；发大乘心，普济一切，愿代众生受无量苦，令诸众生毕竟大乐。

    如此八事，乃是诸佛、菩萨、大人之所觉悟。精进行道，慈悲修慧，乘法身船，至涅槃岸；复还生死，度脱众生。以前八事，开导一切，令诸众生，觉生死苦，舍离五欲，修心圣道。

    若佛弟子，诵此八事，于念念中，灭无量罪，进趣菩提，速登正觉，永断生死，常住快乐。

    此经思想特色鲜明：前七觉侧重个人断惑修道，体现少欲、知足、精进、智慧、布施、离欲等基本修行原则；第八觉则跃升至大乘境界，强调“发大乘心，普济一切”，展现出自他兼利、悲智双运的圆满精神。

    历代高僧如蕅益大师、印光大师等皆推崇此经，并作注疏开示。在天台、净土、禅宗等各宗派中，亦常作为初学入门与日常省察之要典。现代佛教教育中，《八大人觉经》因其简明扼要、切合实修，仍被广泛用于居士共修与僧伽训练。

    《八大人觉经》虽短，却如明灯一盏，照破无明迷雾，引导行者从觉悟无常始，至普度众生终，堪称佛法修行的浓缩指南。
