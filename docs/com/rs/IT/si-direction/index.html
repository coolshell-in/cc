    找到超级个体的产品方向

    2026-05-16 Atom.X

    Whom 受众对象 #

    你帮谁？题目说明：具体描述你目标用户的职业 / 处境 / 正在经历的困境，越具体越好。「年轻人」「上班族」这类太宽，不算。 示例：工作不满2年、每天被事情推着走、总感觉自己忙但没成长的应届生

    传统私营和国企业的管理团队（董事长、总经理），企业产值和员工有一定规模，跟不上信息化时代步伐，面对同行竞争者和经济环境下行，线下营销和整体营收增长乏力。

    How 解决问题 #

    你解决他的什么问题？题目说明：用他自己的语言描述这个问题，比如他平时怎么跟朋友抱怨这件事的？不是你分析的，是他说的原话。 示例：「我每天都在忙，但一到年底绩效会议我说不出自己做了什么」

    希望拓展线上营收，但还没有IT和网络营销团队，不知道怎么做

    Why 为什么？ #

    凭什么是你？题目说明：从三问中至少答一个：①你自己经历过这个困境并且解决了吗？②你帮别人解决过，有具体的结果吗？③你有让你做这件事成本更低的资源或优势吗？答「是」越多，产品说服力越强。

    1. 我已经服务了数十个中小企业、地方政府、投资者，为企业数字化转型、电子商务和互联网营销、以及团队建设做咨询培训；

    2. 在10年内我做了超过10个网红项目，成功推出业界顶流的产品和服务，因此积累了个人丰富的行业经验和技能；

    3. 我在数家业界头部的互联网大厂的运营管理部门工作过，依然有很多的同事、合作伙伴、老朋友在排名前几位的行业大厂里工作，可调用最顶级的技术研发和运营管理专家资源网络；

    What 产品名称 #

    按公式填写你的产品名：【对象】+【时间或数量】+【核心结果】+【产品形式】 示例：全职宝妈·90天小红书从0粉到变现陪跑计划

    传统企业3个月从0开始数字化转型、个人成长和团队建设计划

    How 背景情况 #

    过去 1 年，你心里最想做但一直没做的一件事是什么？是什么阻止了你？现在回头看，你怎么想？

    训练那些生活混乱的人，回到普通人的正常生活方式，减少对其他人和社会的负面干扰；辅导企业主和管理团队，依据商业市场客观规律、ESG标准和当地法规，做有实际成效的工作，告别不切实际的幻想；

    但，大多数人都不愿意付出成本，或早已放弃了改进，成年人世界大概率只能筛选，而非改进，如果想要通过该业务获得收益，势必要扩大受众流量才可能筛选出足够多的客户，但目前我无能找到足够多客户。

    If 愿景假设 #

    假设突然你有一年不用上班，吃喝完全不愁，你最想用这一年做什么？这一年结束后，你希望自己变成什么样？

    继续闭门安静读书升级认知，强化数学和计算机基础，完成至少1个业务模型的验证，并找到一群合作伙伴，1年后能够顺利启动项目，并走上良性增长的轨道。

    Which 厌恶状况 #

    你最讨厌别人对你说哪一句话？为什么？最近一次听到是什么时候？听完你心里第一反应是什么？

    我最讨厌别人说“这也不行，那也不行”，通常这种表达习惯的人已经堵死了发展改进的各种可能，他们的状态很消极颓废，对工作都带着本能的排斥和反对，昨天我还听到这样的话，是我服务的一家教培机构的老师说的，当时听到我的心里第一反应是无语。

    事后我想 “这类自我设置障碍的人太多了，他们只能老老实实一辈子当个接受别人指令的零件，无法独立自主思考和做事，就这种思维惯性，最好还是别误人子弟当老师，此乃国之不幸”。

    When 睡前思绪 #

    晚上睡前你最常想的一件事是什么？想完之后你怎么处理它？这件事白天会想吗？

    睡前没有想什么，只想着睡觉，每晚躺下就打开俄语播客，听着一会儿就睡着了，白天只想当前发生的事，日常有记录，10年前的事都有迹可查，所以睡前清理脑子（缓存），不会隔夜装太多东西，日常鸡毛蒜皮的小事存多了会干扰人，所谓庸人自扰，脑子放空最好。

    电脑和人脑的工作原理类似，如何高效管理资料文件和缓存，类似的方法也适用于人类大脑。

    关键卡点 #

    你目前在副业（创业）上最大的卡点是什么？你为了克服这个卡点都做了哪些尝试？

    要找到足够多客户，必须要足够大的媒体流量，但我并不想做网红，而是偏向于从事幕后技术工作，这个卡点决定了，以上所述的副业我独立跑不通，除非与其他人合作，目前我尚未找到这样的合作伙伴。 为了克服这个卡点，我把周围朋友圈都筛选了一遍，发现朋友圈这有限的熟人都不合适，他们都是更封闭的技术工作者，所以我必须要找陌生人，捷径就是加入某个网络组织，去认识更多陌生人。

    副业优势 #

    你最有把握做出来的副业（创业）方向是什么？为什么你觉得自己会最有把握？（可以包括：过去做过的相关经历、独特优势、其他原因）

    项目经历参考个人网页 https://atomx.cc/zh/，有2个方向：

    1. 教人如何高效率学习和训练良好生活方式;
      我通过网课平台Coursera成功地自学了世界名校的英语、俄语、计算机和数学等10多门课程；我一直保持着很多人羡慕的健康良好的有规律的生活方式；

    2. 农业产业链+互联网的运营管理;

    • 公益：10+家三农和环保类非赢利组织的技术支持；
    • 农文旅：10+个业界知名项目的互联网和电商运营管理；
    • 投行智库：30+家投资基金的行业市场调研和数据分析；
    • 咨询顾问：50+家企业、地方政府的农文旅+互联网咨询；

    轻重排序 #

    如果让你排序，从高到低，你会怎么排序？为什么？（请尽量描述详细）

    • 人是一切成就好事或坏事的核心，这属于思维意识的训练，所以认识人和智慧排第一；
    • 身心健康是人的根基，必须高度重视；人的身心健康的重要程度排第2位；
    • 事业应排第3位，通过互联网做副业是比较轻的创业模式，值得推荐；
    • 洞察宏观经济和市场，对于创业是有帮助的，不过对于某些专注于技术和产品的人而已，他们目标是力争做到行业第一名，可以无视外界环境如何变化，所以这个不是非常重要，可以排第4位；
    • 最不重要的应该是如何打造ip，因为能引起公众关注的明星很少，就像社会不需要那么多科学家和思想家，大多数人都是干手工和体力的，干好了自然就有ip，不需要特意去打造什么ip人设；
