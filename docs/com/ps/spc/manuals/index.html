    公共空间运营手册和实用工具

    2026-04-07 Atom.X

    从业者根据自身所处的工作环境和条件，参照行业标准规范文件，基于“公共空间管理的五大核心原则”（安全健康、提升体验、促进协作、低成本高效、可持续运营）

    制定符合自身情况的日常工作流程和管理办法，该运营手册和实用工具文档，适用于青旅、联合办公、共享公寓、社区中心等各类共享公共空间。

    并持续更新。。。

    《公共空间运营手册》（精要版） #

    目标：为运营团队提供标准化、可复制的日常管理指南
    适用对象：社区经理、管家、前台、保洁、协调员

    安全健康管理 #

    • 每日检查清单：消防通道畅通、灭火器在位、应急灯正常、门禁系统运行；
    • 卫生标准：
      • 公共区域每日消毒2次（门把手、电梯按钮、桌面）；
      • 厨房/卫生间每4小时巡检1次；
      • 床品/毛巾“一客一换”，有记录可查。
    • 应急预案：
      • 火灾：立即启动警报 → 引导疏散 → 报119；
      • 医疗急救：联系就近医院 + 保留现场 + 通知紧急联系人；
      • 冲突事件：隔离双方 → 倾听记录 → 按《用户公约》处理。

    用户体验优化 #

    • 首次接触：提供欢迎包（含地图、Wi-Fi密码、规则卡）；
    • 服务响应：
      • 报修：30分钟内响应，24小时内解决；
      • 投诉：24小时内回访，72小时内闭环；
    • 环境维护：
      • 背景音量≤50分贝；
      • Wi-Fi速度≥50Mbps（公共区）；
      • 温度夏季26℃±1，冬季20℃±1。

    社区协作激活 #

    • 每月活动：至少1场免费社区活动（如电影夜、技能分享）；
    • 协作机制：
      • 设立“共享角”（工具、书籍、食材可登记借用）；
      • 推行“社区积分”：参与清洁/活动可兑换咖啡或折扣；
    • 数字平台：维护微信群/小程序，每周发布活动预告与资源信息。

    低成本高效运维 #

    • 自助系统：
      • 入住/退房全程扫码完成；
      • 水电用量自动计费，账单线上推送；
    • 人力优化：
      • 非高峰时段仅1人值守；
      • 清洁外包+关键点自检结合；
    • 设备选型：优先选择模块化、易维修、能耗低的产品。

    可持续运营策略 #

    • 环保实践：
      • 不提供一次性洗漱用品；
      • 垃圾分类四色桶全覆盖；
      • 鼓励自带水杯（提供免费饮水站）；
    • 财务模型：
      • 收入来源：基础使用费（70%）+ 活动/增值服务（20%）+ 合作赞助（10%）；
      • 成本控制：年度运维成本 ≤ 年收入的40%；
    • 迭代机制：每季度开展用户满意度调研，年度更新《运营手册》。

    🌿《公共空间用户公约》（简洁友好版） #

    目标：明确用户权利与义务，营造尊重、有序的共享氛围
    使用方式：入住/注册时签署，张贴于公共区域

    欢迎您成为本空间的一员！
    为保障每位成员的权益与体验，我们邀请您共同遵守以下约定。这不仅是规则，更是我们彼此尊重、互助与可持续生活的承诺。

    公共空间管理总则：不影响其他人、减少环境污染和浪费、公私分明

    1. 守护安全与健康（安全健康原则） #

    • 不使用大功率电器或私拉电线，离开时关闭个人设备电源；
    • 夜间23:00至次日7:00保持安静，避免影响他人休息；
    • 如感身体不适或发现安全隐患，请及时告知工作人员；

    2. 珍惜体验，彼此尊重（提升体验原则） #

    • 使用公共设施后及时归位、简单清洁（如擦净桌面、冲洗水槽）；
    • 遵守预约规则，不长时间占用公共资源（如会议室、淋浴间）；
    • 交流时友善沟通，杜绝歧视、骚扰或不当言行；
    • 尊重他人隐私，不擅自进入非公共区域或翻动他人物品。

    3. 主动参与，共建社区（促进协作原则） #

    • 参与社区活动或志愿服务，让空间更有活力；
    • 认可他人贡献，一句“谢谢”也能温暖人心。
    • 欢迎提出建议 —— 您的反馈是我们进步的动力！
    • 鼓励分享技能、物品或想法（如借出工具、发起读书会）；

    4. 善用资源，拒绝浪费（低成本高效） #

    • 节约用水用电，离开时关灯、关空调；

    • 自带水杯、餐具，减少一次性用品使用；

    • 看到垃圾随手清理，看到问题主动提醒或报修；

    • 垃圾按分类标识投放（可回收 / 厨余 / 其他 / 有毒害化学品）；

    5. 着眼长远，共担责任（可持续运营原则） #

    • 理解空间需合理收费以维持运营，按时支付费用；
    • 若计划更改，请提前告知，释放资源给新成员；
    • 爱护公物，损坏请主动报备并协助修复。

    🌱 我们相信：
    共享不是“免费使用”，而是“共同创造”。
    您不仅是使用者，更是这个社区的共建者、守护者与传播者。

    感谢您的理解与参与！
    愿这里成为您工作、休憩、连接与成长的美好场所。
    让我们一起打造一个安全、舒适、有温度的共同家园

    社区公约案例 - 简约版 #

    1044栋公共空间运维：

    1. 储物编号：给冰箱格间和储物柜贴房号，根据各自房号存放食品和物品，除非与房主沟通过，否则不可存放物品到在其他房间号中；
    2. 公区共享：给冰箱、橱柜和公区显现位置标明“公区共享”，用于存放可分享给大家的食品和蔬菜；
    3. 恢复状态：使用完厨房和公共区域后，及时清理台面、灶台、清洗餐具、水槽漏斗、抹布等公共物品，保证恢复到未使用前的状态，方便下一个人使用；
    4. 清理物品：设定每周2早上8:00，轮流2人一起清理冰箱和厨房过期食品，无主物品拍照在群里公示，并放入“公区分享”； 5. 物品管理：请告知退房室友，在离开前清理自己的物品，或委托其他人清理，物品要么带走、要么交给管家打包寄存、要么放入公区共享；

    这是最轻量级的公约，务必让每个入驻的成员都知悉，并时常提示周围的人践行该公约。
