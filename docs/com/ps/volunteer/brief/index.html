    劳动换食宿和打工度假网络计划

    2024-11-06 Atom.X

    人类的很多决策都是基于大环境所做出的反应 - 时代背景。

    逆城市化 #

    2010年前后，一系列反应现代城市弊病的网络文学，图书和影视作品发行，例如：

    《逃离北上广》是一套书，说北京太势利、上海太昂贵、广东太竞争。

    《奋斗》：电视剧（也有同名小说），2007年由赵宝刚导演，佟大为、马伊琍、文章等主演的都市青春励志剧。讲述了一群80后年轻人在北京打拼、恋爱、成长的故事。

    《蚁族》：社会学著作 / 纪实文学2009年出版，作者廉思（及其团队）通过实地调研，揭示了聚居在城市边缘、收入微薄、学历较高的大学毕业生群体（被称为“蚁族”）的生存状态。这本书引发了广泛的社会讨论，是“逃离北上广”现象的重要学术注脚。

    《蜗居》：小说改编为电视剧，原著作者六六， 2007年（小说）出版，2009年（电视剧）上映，小说及同名电视剧聚焦上海房价高企背景下普通人的住房困境、职场压力与道德挣扎，尤其是“房奴”现象。电视剧因内容敏感曾一度被禁播，但影响极大。

    《唐家岭》：非虚构纪实作品 / 社会调查报告，作者廉思（与《蚁族》同一研究团队）。唐家岭是北京海淀区著名的“蚁族”聚居村。该书或相关报告深入记录了居住在此的大学毕业生的生活状况，可视作《蚁族》研究的延伸或案例集。

    同样在2010年前后，由于食品安全事件，城市中产阶级兴起下乡开办家庭农场，网络媒体掀起“餐桌自救，做周末地主”的风潮，我也在2010年3月份加入这股返乡种地的大潮中，忙得不亦乐乎，这一系列事件标志着历史车轮行进到一个逆城市化阶段，例如：

    各地农夫市集、CSA社区农场

    以上这些社会事件的高潮已经过去10多年，大家可能已经忘记了，但是还没有，只不过冰山一角露出海面，更大块头还隐藏在水下很深的地方，只要三农发展未取得与城市同等水平，就一直存在势能差的压力，迫使一些人离开城市返回乡村 ……

    这符合物理学的定律，一个U形管倒入液体，两边一定是对齐的，如果某一边加入更多液体，由于压力差，势必会让另外一边也上涨，直至两边压力差平衡，除非中间有某个闸门调节，阻止压力的传输，水库大坝设计要参考这个压力值的峰值，也就是库容量和水位高度，否则大坝存在巨大的安全风险。

    城市vs农村 #

    城市与农村各有优势劣势，对于今日的农村而言，已非同往昔，越来越多的农村房屋宽阔甚至大量闲置，生态新鲜的蔬菜瓜果随手可摘，利用这个特别优势，刚好可以做点什么。

    维度城市农村
    优势
    经济与就业经济发达，就业机会多拥有闲置土地和剩余劳动力
    居住与空间——房屋宽阔，居住空间充裕
    环境与资源——自然生态环境良好，物产丰富且品质较高
    文化与教育教育资源优质，文化生活丰富——
    劣势
    社会问题“城市病”突出（拥堵、快节奏等）年轻人外流严重，人口结构失衡
    居住成本房价高，生活成本压力大——
    工作与心理工作压力大，竞争激烈就业机会少，收入渠道有限
    健康与安全食品安全风险较高，环境污染较严重——
    教育与文化——教育资源匮乏，文化娱乐活动稀缺
    人口与陪伴——留守老人、儿童、妇女普遍，缺乏陪伴与照护

    乡村功能 #

    中国的乡村就像一块巨大的海绵水池，当城市发展需要时，从农村源源不断挤出水分供给城市和各行业发展所需，最重要的农村资源是培养了10多年的年轻人力资源。当城市不堪重负出现产业和人力资源饱和时，或者因为农村建设需要时，又从城市排出大量人力资源，让他们流回农村，这个巨大的海绵水池有这样的泄洪抗旱能力。

    近代历史上出现过的“知识青年上山下乡”运动，有其特定时代背景，其根源也在于我们三农的任务太重太艰巨，离现代化的距离太大太远，单靠农村的内因是不可能实现的，没有一些非常举措，诸如大力动员城市的力量各行各业的支援等，是不可能实现的。

    20世纪50-70年代，我们维持了领土边防和国家安全，以及基本的工业化；80年代至今花了30多年，以城市和工业发展的战略优先，因此我们至今还没有全面完成三农现代化，历史发展到支持三农全面建设正当其时。

    国家政策 #

    中共中央 国务院关于实施乡村振兴战略的意见

    到2018年，中国政府已经连续15年阐述对三农重视的决心和行动。

    • 双创

    中国政府，李克强总理以及新华社等媒体反复报道支持“大众创业，万众创新”。

    即使在改革开放近40年后的今天，许多人还在想着公务员和企事业单位的铁饭碗，尽管政府一再提倡双创，或许因为“脑袋和人生体验”的贫穷“限制了想象力”，雷声大雨点小的双创效果并不明显。

    根本问题是，现有的企事业单位已经无法提供足够的工作岗位了，必须另外想办法自己创业。

    2018年全国大学应届毕业生再创新高达到800万，还有许多往届生处于失业状态。人力资源社会保障部的压力较大，这么多人该怎么安排？广袤的农村大地欢迎大家。

    产业发展 #

    城市产业资本纷纷跨界踏足农业，宣布要投资三农，尤其众多互联网巨头曾大规模在乡村刷墙广告。

    • 京东：发家致富靠劳动，勤俭持家靠京东。
    • 淘宝：生活要想好，赶紧上淘宝！
    • 腾讯：母猪产后护理难？问腾讯元宝！
    • 苏宁易购：苏宁易购，服务到村！
    • 一亩田：用一亩田，卖菜不用愁！（农产品B2B平台）
    • 赶集网：赶集网，找工作、卖东西、找对象！
    • 58同城：58同城，一个神奇的网站！（在乡镇刷得极多）

    由于交通运输、物流、互联网的便利，越来越多的数字游民可以流动到任意一个偏远地区体验生活，这也是我们更多年轻人选择通过互联网创业的原因。

    当政策，资本和市场具足，且中国越来越多的农村地区已经在基础设施（水，电，网，路，快递）等方面条件日益改善，许多地区不比城市差，如果三农整体现状还没有较大改善，只能说明当下最缺现代三农人才！！！

    目前类似wwoof等义工平台，只有简单的线上自助申请对接双方用户的功能，用户量将倍增的关键是增值服务，包括：签证，保险，租车，交通补贴，免年费，多语言版本等。

    还有包括更重要的基于三农的创业孵化服务，这些增值服务，已经有不少人愿意尝试为此买单。

    创业孵化 #

    平台推出创业者孵化群组，招募各种创业孵化器入驻，从平台用户中招募创客加盟，平台提供与政府，地产商，公益基金会，投资基金以及各种资源对接，帮助孵化器和用户实现创业者的聚合。侧重点是做三农，公益平的双创活动支持。

    鼓励大学生和社会青年通过平台组团创业，提供创业孵化所需要的培训，资金，甚至创业园后勤保障服务等。

    最近5年城市科技园从事双创尚未成熟，而各地农村地区又开始做“双创”，换了一个名字叫“星创天地，农创客”，其本质是一样的，盖了一堆楼，却没有好好利用，依然是地产模式，政府买单让我们去做农业电商培训，只有培训机构和运营商赚到钱。

    一批比较精明的电商运营服务商在各地开始接管政府主导的双创园，将以前山寨的“淘宝城”招牌换成了“星创天地”，又套了一笔补助款，但创业效果并不明显。

    因此我经历不同的地区和案例之后，得出一个结论是，可能需要一些公益组织，以及尚没有明确利益动机的志愿者去参与三农的双创，或许会产生不一样的结果，因为他们没有强烈利益刺激，社会理想抱负和名声大于赚快钱的想法。

    媒体铺天盖地报道，热闹一阵后，就剩下一地鸡毛，这些热词逐渐就冷却下来，过一段时间又换成一个新词包装重新粉墨登场，如此循环往复，这不禁让人感到无奈和忧伤，社会资源就是这么被大量浪费掉的，真正做事的人，你不知道他们在哪里，他们也往往得不到帮助，而那些投机取巧挥霍无度的人，惯用套路一次次得手。

    社会效益 #

    尽管现实有种种不堪，但我们还是要往前走向远方看，为了说明平台如何运作，列举该现实案例。

    冰花男孩 #

    “冰花男孩”是中国一万网友关注的一位留守儿童形象代表，其故事曾引发全国对乡村教育、贫困儿童和留守儿童问题的深切关注，以及关联的偏远地区的三农发展问题。

    新华社： “冰花男孩”意味着什么？

    由于地处偏远且较为落后，当地缺资金，不可能有钱支付工资请人过去工作，只能招募义工协助当地至少3个方面的建设：

    经济

    扶贫：优势农业资源，例如邵通苹果

    机构：对接销售渠道，例如e农计划，诚实善粮等公益组织和慈善企业，以及其他常规流通渠道；

    功能：帮助偏远农村地区销售优质农产品，提高贫困农民收入，唯有让爸爸妈妈回家，山村的孩子将不再孤独留守。

    教育

    支教：山村孩子的守护天使

    机构：对接支教公益组织，例如天使支教，以及其他自然教育（非应试教育）资源；

    功能：为乡村儿童提供长期成长陪伴服务，进行教育创新探索（创造力培养与归属感建立）的公益项目。

    环境

    环保：减少农药化肥污染

    机构：对接环保技术，例如云南思力生态替代技术中心，以及其他环保产业技术；

    动作：提高农业劳动生产效率和产品品质，保护环境，不以牺牲环境为代价换取短期的发展利益。

    我们可以通过本义工招募平台，为该地区提供源源不断输送各种专业技能人才，以及创业孵化，公益支持。

    任之堂 #

    任之堂是中国当代颇具影响力的民间中医传承与临床实践平台，由中医师余浩（网名“任之”）于2000年代初在湖北十堰创立。

    完全通过志愿者改造抛荒20年的乡村，变为网友热捧的中医村。通过中医诊所、任之堂养心山庄，弘扬中医文化，宣传养生理念！

    它以“回归古中医、重视脉法、强调临床实效”为特色，融合传统师承教育与现代传播方式，在中医爱好者、从业者及患者群体中拥有广泛声誉。

    堂主经常每发一篇文章，都有数千上万的访问量，每次活动动辄数十乃至数百的报名。

    其他案例 #

    • 朴和素

    位于杭州富阳龙门镇，该家庭农场农场面积很小，从来都不做商业推广。房子小而且破，冬天漏风，条件不好，但依然有不少外国志愿者申请，经常要拒绝掉很多人，几乎每个周末都有各地的朋友来探访农场主，因为他坚持很多年不用农药化肥，很多城市人愿意开车过来购买生态蔬菜。

    • 无畏护生园

    该庇护农场位于河南焦作武陟，1200多亩，生活着60多种动物，于2014年创办，焦作市动物保护救助协会筹建，河南泰鑫农业有限公司捐赠土地使用，旨在保护救助被遗弃和被暴力伤害的流离失所的无助动物。

    目前该基地的资金主要来源于释妙无法师自己的积蓄，以及来自各处的善人捐款；同时吸纳许多网络上对农场动物保护感兴趣，或想尽自己一份力的志愿者团队。
