    乡村建设团队的文革情结

    2013-09-20 Atom.X

    三农和乡村建设组织是培养狭隘的极左民粹的大本营，21世纪文革愤青延续的精神根据地。

    激起愤青 #

    温教授有一本书叫《告别百年激进》，核心是反思西方主导的 “激进现代化”路径，纵观三农理论和乡村建设的历史事件，充满灰暗和凄凉，事实证明，他并非温和，反而是非常激进的，极左或极右导向明显。

    我们加入三农和乡建组织，一起在小毛驴农场同吃同住同劳动的经历，体验到最明显的变化是：“我们变得更愤青了”。

    这些组织的特有行为方式，似乎暗示文革尚未过去，也不是在真正搞三农发展和乡村建设，而是在利用不谙世事的学生搞社会运动，似乎有某些特别动机。

    很多人投入1-2年甚至更长时间来做义工，最后却选择远离他们，因为学生们没有成长收获，没有学到什么有益的知识、技能或精神思想，这些组织的导师们起着负面的示范作用，展示很多负面的东西，却没有展示多少优秀的一面。

    “聚是一团火、散是满天灰”，不是温教授说的“散是满天星”，也不是蒲公英飞到各地播种，每次散伙都留下一地鸡毛和狗血，这就是当时我们经历的三农和乡建组织的真实状况。

    说一些具体的经历吧！

    唱红打黑 #

    2012年初刚过春节，小毛驴农场的很多同学都没回来，我们觉得挺奇怪的，为啥农场一下子空了？

    原来他们被乡建团队带到重庆，参加“唱红打黑”活动去了，重庆出钱让学生们去站台。然后不久薄熙来倒台，他们也在媒体上沉寂了一段时间，那时我已南下杭州工作，这中间发生了什么，不得而知。

    温教授喜欢唱红歌，每次在农场聚会，他总是特别兴奋地带头唱，三农乡村建设核心团队有几个人也擅长唱红歌，把人选派到重庆能发挥特长，可以再多开几个乡建基地。

    温教授为代表的那批50年代出生的人，正是年轻气盛的时候，经历过完整的文革和知青上山下乡运动，结果造就了两类人。

    80年代流行伤痕文学，很多知青写下充满精神创伤的回忆录，对那段经历感到厌恶和憎恨，言语总是戴着戾气和不安，因为他们家破人亡或终身严重伤病，留下文革后遗综合症，很多80后出生的人不知道这段历史的细节。

    另一类人很怀念那段激情燃烧的岁月，退休后还经常聚会，一直到老了还充满亢奋，跟随媒体的节奏喊打喊杀，他们不屑“和气生财、韬光养晦”这种说辞，只有斗争和对抗的路可走。

    同在重庆的还有另外一位名师 - 北京地球村环境教育中心的廖晓义，是中国知名的环保活动家和生态文明守护者，我参与过她在巫溪县做的乐和家园，同样也是乡村建设项目。

    廖老师也是50年代生人，同样经历过文革和知青运动，她也喜欢红歌和革命样板戏，比温教授行为更过激的是，她在千人国际会议上激烈批判（大骂）某位嘉宾，面对面地毫不留情地、怒目而视，或当众哭泣大喊大叫（歇斯底里），因为她对相关方感到失望或绝望。

    在西南大学乡村振兴战略研究院的官网上，戴锦华教授和廖晓义老师也被列入特邀研究员。

    我周围有很多生于50年代的长辈和老师，具备以上两种典型特征，上面几位老师大概都属于后者。

    红卫兵 #

    2010.6.12，温教授59岁生日。地点：梁漱溟乡村建设中心，北京市海淀区温泉镇温泉村。

    温三农旗下的乡村建设各团队的上百名学生来给教授庆生，作为小毛驴农场的一员，我也获邀参加。

    走入活动现场大院，到处都是红旗飘飘、大字报、大横幅、学生们穿着绿军装、红五角星帽子、带着红袖标，拿着红缨标枪列队在道路两旁，热烈欢迎温教授和来宾，并给温教授系上红领巾……

    我当时有一种穿越到文革年代身临其境的感觉，温教授就像是伟人毛主席，只有在电影、电视、图片中见到过这样的震撼场景。

    如果有导演想拍革命题材和样板戏，强烈推荐去梁漱溟乡村建设中心，我们一定会帮忙联系沟通，上哪里找这么好的群众演员和主角？

    温三农的乡村建设团队，用实际行动数十年如一日地坚持每天实地演练，才会自然而然达到电影拍摄场景的效果，根本不需要上电影学院和学造型艺术设计，他们天生就会这套功夫，这是天赋。

    曾经在中文搜索引擎baidu上检索“三农、乡村建设、支农、大学生、志愿者、下乡”等关键词，也能看到几乎一样经典的照片，红卫兵的装扮包括：大红色主题、旗帜、帽子、红领巾、红袖章、服装和标识物，这和50年前的红色革命几乎没差别。

    乌有之乡 #

    我们很多人的时政启蒙是从小毛驴农场（三农和乡建）圈子开始的，从那里我们找到知名的中国左派大本营“乌有之乡”网站，2012年在乌网上，经常能看到温教授的文章，但同时似乎又有很多内容，让他表现出右派的特点，这让人搞不清楚温教授的倾向和立场是什么。

    小毛驴农场区别于一般的农场，有个重要特点是强调三农和乡村建设思想的学习。

    温教授经常邀请一些同事和关系密切的学者，来给农民和学生上课，因此我们有幸能接触到许多知名学者，三农和乡建圈子里聚集着大批温教授的左派死党。

    我们在农场干完一天活儿后，经常在晚上集体讨论学习，可能会邀请一名老师过来，给我们讲某个主题。有看影片、读书会等不同方式，看完后大家交流，老师引导讨论和思考。

    切. 格瓦拉 #

    在诸多学者中，北京大学戴锦华教授频繁受邀来开讲座，她与三农和乡建圈子保持非常密切关系，其他人很难接触到这些知名学者，但是通过小毛驴农场，我们似乎非常容易有各种机会见到名人。

    第一次见她，是2010年农场收工后秋日的夜晚，在农场实习生的宿舍里，有大约不到20个人围坐一起，例行开会学习，她给我们讲古巴共产党领导人切. 格瓦拉，若论电影纪录片和人物传记，中国可能没有几个人比她更资深的，当时在座的大多数新同学是头一次听说格瓦拉，也不知道面对的是怎样一个来自北京大学的知名文化专家。

    当时，她一边讲一边叼着烟大口大口地吸，严晓辉也和她共同陶醉在烟雾中，毫不顾忌正面对着的是一群涉世未深的学生，这是我头一次见到这么豪放不羁的讲课方式，由于思想锐度和特立独行风格，颇有北京爷们风范，因此她被许多北大的学生尊称戴爷。

    后来我们知道，此二人经常开会时的间隙扎堆抽大烟，戴老师和学生小严的关系可不一般。

    在一片烟雾朦胧中，讲到最感动的地方，戴教授深情地称“切是她的梦中情人，在西班牙语中，只有情人才会亲昵地称呼切”，这是她的原话。

    她称赞格瓦拉是优秀的无产阶级革命家和战士，将格瓦拉被中情局枪杀，比喻成耶稣受难，格瓦拉的遗容就像耶稣受难后一样圣洁。了解世界历史和政治的朋友知道，“切. 格瓦拉”象征着红色革命。

    后来，我也看过一些其他来源的资料，有人指责格瓦拉是战争屠夫和杀人狂魔，譬如高晓松的节目中就说过“格瓦拉是个大妄人，周恩来对他的评价很低，左翼青年盲目崇拜格瓦拉，搞个人英雄崇拜”，发出这两种截然相反观点的都是我们尊敬的大V。

    由于世界历史和人文学科知识匮乏，我无法判断到底哪一方说得对，仅仅从小毛驴农场和乡建圈子里的格瓦拉赞美者的日常行为中观察，很多人表现出狭隘的人生观和世界观、政治极左和极端民粹的特征，为何不其他人列为崇拜对象？这大概就是三农和乡村建设思想基础的必要组成部分。

    回到初衷，我们大多数人当时去小毛驴是为了学习经营生态农场，回去创业做健康农产品，保障自家人的食品安全，赚点钱并顺便为社会做点贡献，结果却让我们学习左派的红色革命思想？好像车子开错了方向，不对啊，方向盘撸错，被人带偏了。

    也有一些同学头一次听到格瓦拉的讲座很兴奋，在讨论时，觉得就是要干死美国佬，才能世界和平天下大同。但通常热情维持不了多久，该干嘛得干嘛，还是挣钱吃饭要紧，谁没事会去搞美国佬？穷得口袋比脸干净，连张跨越太平洋的机票都买不起，没那个本事。

    梁漱溟乡村建设中心与小毛驴农场是并行的兄弟单位，同为三农和乡建理论培训基地，全国各地的三农和乡建社团都来取经，其中东北地区的三农社团做得很极致，包括大连理工大、鞍山师范学院、东北师大沈阳理工等10多个高校，2007年一次将近100人的培训会上以“切.格瓦拉”作为衣服的标识和社团主题曲。

    可能大多数的80后不知道格瓦拉是谁，但你可能见过他，在中国各大旅游区的步行街，总是能找到格瓦拉主题的各种文化商品的小店，格瓦拉的风靡程度与毛主席几乎是齐名的。

    很多搞摇滚乐的人，都喜欢穿戴格瓦拉主题的文化衫和各种摆设，那代表一种左派先锋的思想。不了解的人，以为文化衫上印的那个头像带着贝雷帽的小哥（格瓦拉）是唱摇滚的。

    我曾经有大量外事接待工作，见过一些知名人物，2011年冬天，作为三农和乡建体系一名工作人员，我服务南南论坛，接待并陪同十多个国家的社会学者访问江浙沪地区，包括阿莱达. 格瓦拉 - 切. 格瓦拉的小女儿。

    她参与国际活动的身份是医生+社会活动家，是古巴儿童医院的一名医生、古巴红十字会、泛美卫生组织合作专家或公益组织代表，可持有公务护照，在机场走外交或绿色通道。

    作为格瓦拉的后人，出生背景特殊，但她对政治舞台很淡漠，也不持有某种意识形态立场，这是我亲自和她对话后的感受。在逛杭州河坊街时，我买了一个格瓦拉主题的钱包送给她，为了离别前的纪念，不知她感受如何，印着她父亲头像的小商品到处泛滥，无人维护和规范其肖像权。

    偶像崇拜 #

    戴教授对切·格瓦拉（Che Guevara）的崇拜是众所皆知的，这位20世纪拉丁美洲革命家对她的精神世界和学术研究产生了深远影响。以下是具体细节：

    • 精神偶像与精神危机中的相遇

    戴锦华在1990年代经历了一场严重的精神危机，感到学术与个人方向迷茫。正是在这一时期，她通过研究“全球六十年代”和第三世界运动，深入了解了切·格瓦拉的生平与思想。她形容这种相遇如同“生命中的一次恋爱”，并称格瓦拉是“我们时代的完人”（引自萨特的评价）。

    • 人格魅力与精神高贵的象征

    戴锦华被格瓦拉“高贵的背叛”所震撼 —— 他出身阿根廷富裕家庭，却选择为无产阶级和第三世界的解放而战，最终牺牲于玻利维亚。她认为格瓦拉展现了超越“趋利避害”动物性的人性光辉，是“舍生取义”的典范。她甚至拒绝观看任何关于格瓦拉的影视作品，认为演员无法还原其人格魅力。

    • 学术与行动的启发

    格瓦拉引导戴锦华重新思考全球60年代的反抗运动（如民族解放、女权运动等），并推动她实地考察第三世界国家。她曾专程前往古巴圣克拉拉的格瓦拉墓地，泪如雨下，感受到一种“精神原点”的力量。她将格瓦拉视为对抗时代荒谬与精神贫瘠的象征，其名言“做现实主义者，求不可能之事”成为她的座右铭。

    • 书房中的“神圣位置”

    在戴锦华北大办公室的软木板上，格瓦拉的照片与戈达尔、玛丽莲·梦露等并置，占据显著位置。她直言：“我可以跟人心平气和讨论任何问题，但不跟任何诋毁切·格瓦拉的人做任何讨论。”，格瓦拉是她跨越时空的精神偶像。

    看到这里，我不得不说，古今中外历史上的那么先贤，不值得学习吗？怎么就找到一个极其偏远的南美洲的革命党偶像，一上来就痛哭流涕直接感动得见到上帝似地，社会上类似的人蛮多，各自崇拜的对象都不一样，这不就是追星族们吃饱了撑的吗？都怪袁隆平爷爷，粮食太多也是个问题。

    与温教授类似的背景，她出生于1950年代北京知识分子家庭，经历过文革和知青运动，父母从事教育与文学领域，家庭环境对其学术道路和兴趣起到重要作用，北京大学毕业，成长环境优渥，养尊处优，喜欢胡思乱想，女性主义学者，未婚却经常能把自己感动得怀孕。

    反西方 #

    温教授经常讲“反思现代”和提倡“生态文明”发展，这些理论应该是一种“包容、多元”的精神，他还明确说“让学生们回到农村去，发展一个本地化的内生系统，警惕外来的西方思想和资本主义”。

    在2010-2012年前后，学生们对温三农理论的印象就是警惕西方，旗帜鲜明地反西方的立场，这或许是学生们普遍存在的误解，不知道这个误解时怎么奇妙发生的。

    总之我和学生们都普遍理解温教授的核心思想是：反西方、反现代化

    我们在不同场合听到温教授发言的不同版本，他一直在反复告诫学生们一些严重问题，反对清单主要包括：

    反四化：反对 “私有化、市场化、自由化、全球化”

    后来从温三农团队传递出来的“反对”清单扩展了很多，包括并不限于：

    反八化：反对 “商品化、现代化、工业化、城市化、资本化、国际化…… ”。

    温教授的乡村建设的组织体系自上而下，包括小毛驴的管理层，都透露出这种“反四化、反八化”观点。

    西方宗教 #

    在一次公开讲座或访谈中，温教授关于中西文明差异、宗教一元性与多元性以及其对农业模式影响的论述，这类观点常见于他近年来强调“去西方中心主义”“生态文明”“在地化发展”的演讲中。

    其中提到西方宗教，温教授原话大意如下：

    从事三农工作的年轻朋友的困惑，这属于中国现有主流的教育和思想体系中所贯穿的“西方中心主义思想体系“，在漫长的黑暗的中世纪只许信一个教。因为他们太散乱，谁都管不了谁，而主教是精神权威，国王、贵族认命，只信一个神，是一元的体系。而中国什么神都信，中国是多元化，所以中国适合做生态农业。

    没找到当时讲话的这篇原稿在哪里，知道的朋友请找到网页链接并分享出来。

    连西方宗教也一起反了，还有什么不在反对的清单里，这变成了彻底反西方文明和意识形态，愤青几乎都是反西方的，他不就是愤老么？

    殖民历史 #

    温教授经常喜欢讲：殖民历史、漫长的黑暗的中世纪，在小毛驴的读书清单里，我看过一本《拉丁美洲被切开的血管》。

    他总是替第三世界国家的殖民历史哀叹，这种特殊的情感是怎么来的？

    反智言论 #

    温铁军：美国金融资本那么强大,为什么无法振兴实体产业？

    摘录其中一句话：

    “但是70多年过去了，如今的美国却不再是当年的世界工厂。原来强大的工业生产巨擘，如今为何却连一件像样的工业品都没有能力生产。”

    什么叫无法振兴实体产业？发达国家把很多产业都转移到发展中国家，不屑于开高耗能污染低附加价值的工厂，这确实是没有生产了。

    什么叫“像样的工业品”？航母、NASA空间站、Space X火箭，算不算像样的工业品？

    让人惊掉下巴，难以置信，突破了正常人的认知底线，真敢于这么胡说八道。

    这种直白地反智商的文章，确证是教授的原话，要么是年纪大了头脑发昏，要么是装糊涂，有意地在收割极左民粹网友的智商。

    对冲和投机 #

    为了更好理解这种奇特的操作手法，借用“对冲基金”解释一下。

    了解一点金融的朋友都清楚，对冲（hedging）的核心思想是“两边都投、平衡风险”，目的是降低风险，防止亏大钱。同时持有方向相反、相关性强的资产，让一个亏时另一个赚，从而抵消部分或全部损失。

    经典例子： 农民担心未来玉米价格下跌 → 在期货市场卖出玉米期货； 如果到期价格真跌了，现货亏钱，但期货赚钱，整体收入稳定。

    但“对冲基金”这个名字有误导性 —— 它不等于“安全的投资”，反而是更高风险、高门槛、高费用的代名词。

    对冲特点包括：

    • 目标不是“保本”，而是“绝对收益”：不管牛市熊市，都要赚钱（甚至熊市赚更多）；
    • 策略极其多样：多空股票（买好股+卖空差股）、全球宏观（押注汇率、利率）、量化交易（算法高频买卖）、并购套利、事件驱动等；
    • 常用杠杆和做空：可以借10倍钱炒股，也可以“先卖后买”（做空），放大收益也放大风险；
    • “对冲”可能只是辅助手段：很多对冲基金并不真正对冲风险，反而主动承担高风险博取高回报

    现代对冲基金更像是使用复杂工具、追求高回报的精英投资机构，名字里的“对冲”更多是历史遗留。

    如果你听说某人投了对冲基金，别以为他在“避险” —— 他很可能在主动拥抱高风险，只为博取超高收益。

    正如金融界常说：“Hedge funds don’t hedge — they speculate.” 对冲基金不对冲，它们是在投机。

    所以，不要顾名思义，而要根据其实际行为判断其本质

    行胜于言、行动比语言诚实。
