## Claude Certified Architect (CCA-F)

The Claude Certified Architect – Foundations (CCA-F) is Anthropic’s foundational-level certification validating the ability to design, build, and operate production-grade AI systems using Claude, Claude Code, and the Model Context Protocol (MCP).

## What is CCA-F?
The CCA-F certification, launched on March 12, 2026, is a professional credential from Anthropic that focuses on applied AI architecture rather than simple API usage or prompting. It validates that candidates can architect reliable, safe, and performant AI systems using Claude models, MCP, and Claude Code across production workloads, including agentic loops, structured extraction, code generation, and multi-agent orchestration 

Unlike academic or cloud AI certifications, CCA-F emphasizes real-world system design, operational reliability, and scalability.
