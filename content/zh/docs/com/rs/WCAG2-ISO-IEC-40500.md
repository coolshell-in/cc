## 🌐 **WCAG 与 ISO 标准的关系**

该页面是 **W3C Web Accessibility Initiative **(WAI) 官网中关于 **Web 内容无障碍指南**（WCAG） 的权威说明部分，特别聚焦于 **WCAG 与国际标准化组织**（ISO）。

### 核心内容要点：

1. **WCAG 是什么**？  
   - WCAG（Web Content Accessibility Guidelines）是由 W3C 制定的一套技术标准，旨在让残障人士（如视障、听障、运动障碍用户）能够平等访问和使用网页内容。
   - 当前主流版本为 **WCAG 2.1**（2018年发布），**WCAG 2.2** 已于 2023 年成为正式推荐标准。

2. **WCAG 与 ISO 的关系**：  
   - **ISO/IEC 40500:2012** 是 **WCAG 2.0** 的正式国际标准版本。
   - 该标准由 **ISO 和 IEC**（国际电工委员会）联合发布，**完全等同采用**（identical adoption）W3C 的 WCAG 2.0 技术内容，未作任何修改。
   - 这使得 WCAG 2.0 获得了全球政府、采购机构和法律体系的正式认可（例如用于公共采购或立法引用）。

3. **为何重要**？  
   - 虽然 W3C 本身是非营利标准组织，但通过 ISO/IEC 认证，WCAG 被纳入**正式的国际标准体系**，增强了其在政策、法规和跨国项目中的权威性与可执行性。
   - 各国政府（如欧盟、加拿大、澳大利亚、日本等）在制定数字无障碍法律时，常直接引用 **ISO/IEC 40500** 或 **WCAG 2.1/2.2** 作为合规基准。

4. **后续版本是否纳入 ISO**？  
   - 截至 2026 年，**WCAG 2.1 和 2.2 尚未被 ISO 正式采纳为新国际标准**，但 W3C 正在推动这一进程。
   - 实践中，**WCAG 2.1 AA 级已成为全球事实上的无障碍合规基准**，即使尚未有对应的 ISO 编号。

---

### ✅ 总结一句话：
> **ISO/IEC 40500:2012 = WCAG 2.0 的国际标准身份**，使 Web 无障碍指南获得全球官方认可；尽管更新版本（2.1/2.2）尚未完成 ISO 流程，但已被广泛视为现行最佳实践。
