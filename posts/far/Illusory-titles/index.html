    如何构建“三农问题专家”的超级IP现象

    2012-03-18 Atom.X

    只要提起三农，指代的就是主流媒体上的明星和网红 - 温铁军教授，他确立了“三农问题专家”和“乡村建设”的身份标签，这2个概念是他所属的专有名词。

    我们通过这些大头衔简单地分析一下，如何构建“三农问题专家”的超级IP现象。

    头衔解析 #

    温教授在媒体上的头衔很多，包括：著名的三农问题专家、教授、博士生导师、农业经济学家，领国务院津贴的学者等等。

    相关背景信息：《 乡村建设、三农问题的历史源流》

    在诸多头衔中，“三农问题专家“ 最深入人心，几乎所有主流媒体如新华社、央视、《人民日报》对其报道均使用此称谓。

    细品“三农问题专家”这几个字，你会发现这个头衔确实很耐人寻味。用修辞学（Rhetoric）、考据学（Textual Criticism / Evidential Research）、说文解字（字源/字义训诂）的方法，把这几个字拆开，不同的排列组合，一一解读。

    独家、唯一性 #

    “三农问题专家“或”三农问题专家“大头衔，独此一家，别无他人，这是媒体和公众对温教授的认知，也是他公开接受的身份标签，这个IP打造得非常成功。

    李昌平老师也有挂这个大头衔，但是流量很难上规模，无法在公众心目中构建一个独一无二的IP角色，尽管他起步更早，并首先提出“三农问题”这个关键词，被媒体和国家政策层面所广泛接受，但很快这个概念一哥的角色就转手让给了温教授，李老师的IP运作不给力。

    “专家”指代的是在特定领域内，通过精深的系统知识、长期实践与社会认可，具备解决复杂问题能力并承担相应责任的权威实践者。

    在任何行业领域，可能同时并存无数的同等职称的专家，同行通用同一个头衔，可能行业会议上同时有100个人都叫计算机安全专家。

    但中国挂有“三农问题专家”头衔的，唯有此二人，公众基本忽略李老师，而专指温教授，因此这个头衔具有独家、唯一性、排他性。这是非常值得探究的一种奇特现象。

    “三农”专家 #

    “三农”指代三个方面：农业、农村、农民

    这三个概念中的每一个都是非常大的，其内涵和外延的范围非常广。

    为了方便分析，去掉“问题”二字，只叫“三农”专家，无论要不要在专家头衔前加“问题”二字，“三农”的概念依然太空泛了。

    农业分类 #

    为了深入解析，先选取其中一个“农业”概念论述。

    • 行业维度

    依据《国民经济行业分类（GB/T 4754—2017）》，“农业”属于A门类，包含“农、林、牧、渔、专业及辅助性活动“5个大类、25个中类、68个小类；

    涉及的物种大类（非细分小类）包括，50+种常见经济作物、20+种主要畜禽及特种经济动物、上百种经济水产动植物，这已经太多了。

    还不包括成千上万种的农产品初加工与辅助活动的类别。

    • 职业维度

    依据《中华人民共和国职业分类大典（2022年版）》，包括农、林、牧、渔业生产及辅助人员5大类，分为5个中类的生产人员，包括：农业、林业、畜牧业、渔业、以及“农、林、牧、渔业”的生产辅助人员。

    还包括30个小类，农业领域共有约 113 个官方认定的细分职业工种。

    以上所列的总计有数百上千种“农业”相关的行业和职业，做一个总包的“农业”专家，谁够资格当这样的专家？没人敢回答这个问题，温教授做到了，而且总包的范围更大到“三农”，而不仅仅是“农业”部分，名正言顺，一当数十载。

    还有一个中国农业大学，里面很多专家学者，他们情何以堪？这个头衔被你一人包圆了，我们干啥去？

    当我们说“农业”，根据国家相关标准分类，主要指代自然科学和理工科范畴，我们都知道中国农大是以技术为主的专业，温教授是人民大学的老师，属于文人社科类专业，你怎么包场包到了中国农大来了？

    搞技术的人都比较老实，中国农大没人议论这事，这样就显得温教授太狂野和过分了，不把国家教育部门的职能分类放在眼里，他想跨界到哪儿就直接包场过去了，很是潇洒惬意。

    虚空 #

    你是否听说过工业专家、互联网专家、医疗专家？为何农业领域就能出现“农业专家”，而且三个大概念叠加一起，叫“三农专家”？

    受过基础教育的正常人，都知道这些概念有多大，例如“工业”，包含了成百上千个大的门类和细分的职业工种。

    “农业”和“工业”概念的体量是对等的，两者涵盖的范围都非常大，既然不能说“工业专家”，为何能说“农业专家”？更遑论更大的“三农专家”？

    正常人的头衔介绍是很具体的，例如我们有计算机安全行业开会，大家都统称叫计算机安全行业的专家，水平当然有高低水平的不同，按照段位分初级、中级、高级等，然后细分到某个具体领域，有操作系统安全、网络金融安全、数据库安全等不同的分类，行业内的称呼都会细分到某个具体的“工种类别+资格级别”的职称头衔，例如“操作系统安全初级专家”。

    又例如说某某是“机床高级钳工”，而不能随便扣一个笼统的“工业专家”大帽子，如果谁敢这样做个人介绍，对方可能会说“啥玩意儿？神经病吧，特么地糊弄我呢！”。

    有人敢为天下先，精神可嘉，同时把（农业、农村、农民）三顶大帽子戴头上的，这是天下第一人，必定承受得起这堆大帽子，可能也很陶醉和享受的，乐于戴这些大帽子，觉得自己很适配。

    这类虚空的头衔有点类似“阿弥陀佛Amitabha”，无量光和无量寿的意思，理想可能是成为圣人、宗教创始人、仙佛、伟人，还不如直接说要创造（三农）宇宙的真理。

    “问题”专家 #

    世上居然有“问题”专家的头衔？这是一种奇特的现象，仅仅存在于三农或农业领域，其他行业没有见过。

    怎么凭着提问题，就成了专家？这专家也太水了吧，谁不会提问题呢？哪一个行业没有问题？

    • 医疗问题

    无数的疑难杂症至今尚未解决呢，你是医学系博士，可以叫自己“医疗问题专家”或“健康问题专家”吗？

    • 工业问题

    中国还有很多关键的卡脖子技术有待攻克呢，你是做工业机床，可以自称“工业问题专家”吗？

    • 互联网问题

    废话，网络安全和网瘾一直有许多问题，你是学计算机的，可以叫“计算机问题专家”或“互联网问题专家”吗？

    你只要敢给自己安个类似以上的某个行业领域+问题专家这种格式的头衔，肯定要被同行或公众喷口水。

    温教授又实现了，尚未被口水淹死，反而在媒体舆情的公众心目中的名声是相当好的，他如何做到的？

    “问题”源头 #

    因为一切问题都是人造成的，当我们说“三农”问题时，应该首先关注人的问题，因此我们找到分析问题的入口，也就是三农中的“农民”问题。

    引用民国时期乡村建设派学者晏阳初的著作，温教授的三农理论中给当代农民打上愚贫弱私（愚昧，贫穷，弱小，自私）的标签，通过乡村建设和三农的宣传渠道持续放大这种印象。

    这个身份标签的打击面太大，我不太同意，并不因为我的出生背景也是农民。

    不分时代背景和地域差异，一概而论所有的农民都是愚贫弱私的特性，这种说法很容易让人断章取义。需要看数据说话，必须具体到某个地区、某个年代、到底有多少比例的农民是这样的特征标签？

    我尊重知识分子，但不轻信权威，需要针对非常具体的案例去探究和理解才可以下结论，此非严谨学术风格的学者所言，因此我保留质疑。

    但很多从事乡村建设的学生和公众已经接受了这一断言，这就是“三农问题”的源头，公众都默认了“三农就是个问题”。说起“三农”好像一直都是问题，似乎乏善可陈，烂摊子简直到了无法收拾的地步。

    如果老是说“三农是个问题”，这确实不厚道，让谁听了都不太开心，就好像总说一个人有病一样，能不能说点农民的好话和优点？

    能做到教授级别，当然是很精明的，不能过度去说“农民”的问题，而更多“设身处地”为农民说话，设立“中国农民的代言人”的形象，经常在媒体上替农民打抱不平，让农民感觉，还是这个专家有良心，是我们自己人，对比发现其他专家都是没良心的。

    这就是戴着不合适的大帽子，却没有被口水淹死的秘诀，其他人都做不到，唯独温教授此一例。

    继续深入分析。。。

    “三农问题”专家 #

    顾名思义，尊重事实，专家本人也如实陈述的，他没有欺骗公众，这就是个 “问题专家”，而非 “解决方案专家”。就是他的本质已经在头衔上说得明明白白，如果有误解，那是公众自己的问题，角色发生奇妙地翻转了，关键转折点就在这里。

    能给“解决方案”才是真正的专家，知识分子提问应该比较有水平，但问题提得再好，不解决问题，光提问有什么用？

    社会需要解决问题，而不仅会提问题。既然一直喊三农是问题，而且是严重的问题，公众期盼专家们给解决问题吧。

    从千禧年到现在，这么多年过去了，“三农问题专家”的头衔在媒体上一直都挂着，没人骂它很虚空，这种现象是很诡异的，本篇算是互联网上首次解析此案例的吗？

    自2010年起始，我们参与小毛驴农场项目有2年，认识温三农团队也有10几年，依然搞不清楚他们有提出过什么好的“解决方案”。

    有一个简单提问方法，只要问他们2句就够了，“你们做三农发展和乡村建设，有什么成功案例吗？那个地方接受你们指导并成功的？”

    由此可见，在中国三农领域，很缺乏有能力和智商的人，所以才让虚妄的头衔劫持了媒体话语权，没人纠正这个问题，连事情“伪造、欺诈”的定性说法都没有。

    要么这个圈子里有人看明白了，但是呼声太小，公众听不到；也有些明白人，假装糊涂，明哲保身，害怕得罪专家。剩下的吃瓜庸众，是真地看不懂问题的本质，跟着媒体乱起哄，把这个问题专家的头衔叫得响亮。

    于是主流媒体上，“三农问题专家”在过去的20多年一直在糊弄着，却被十几亿人默认这个专家就是能解决问题的天命之子，公众脑子真是被洗得很彻底。

    这么一个虚妄头衔，如何又在公众心目中确立了 **“解决方案专家、实践家、实干家”**形象的？

    谎言说一万遍，就成了真理，这是宣传的基本方法，尤其借助互联网力量，只要持续说，就能占领受众的心智，就这么简单的方法。

    所以，要不停地讲，即使年纪很大，也不能退休，必须成为网红。

    懂互联网流量密码的操盘手们都知道，网红和受众之间是弱关系，对于文化IP根基很浅的网红，必须依赖泡沫流量支撑持续的热度。成为网红的基本功必须是非常勤奋，否则很快就“昙花一现”消失了。

    我们要向网红学习，即使没有任何其他优秀品质，成就大事业者至少需要一个重要品质 - 勤奋。

    乡村振兴专家 #

    温教授旗下的自媒体团队，可能自己都觉得这“问题”二字颇为蹊跷，后来悄悄把发布文章中“问题”二字逐渐都给去掉了。

    但“问题专家”头衔过于深入人心，以至于公众短期之内无法消除这一刻板印象，网络上依然有太多的带着“问题”专家头衔的文章，那是别人发的，他们无法屏蔽禁止或阻止这个头衔的持续流传。

    所以在媒体上我们看到，温教授就在 “三农专家”、**“三农问题专家”**两种状态中切换，一会儿有问题，一会儿又没问题，无法给自己一个精准定位 - 他到底是谁？

    再然后，他们可能觉得“专家”二字已经被污染了，他们清楚知道自己就是伪专家的（受害人）代表，公众看了“专家”就想骂，在媒体上普遍不被接受，于是最近在他们发布的文章里的头衔又改叫 **“三农学者”**了。

    现在，国家主流媒体叫“乡村振兴”，于是他们也不再叫了20多年的“乡村建设“，又集体把头衔改成了“乡村振兴专家”。

    目前，温铁军担任西南大学中国乡村建设学院执行院长，由其“乡村建设”原班人马和原三农专家团队主导，聚焦乡村振兴与生态农业。

    善于投机钻营也是重要品质，又让他们捡到好果子，大家一起上，不能好果子都让他们捡走了。

    徒弟继承虚妄头衔 #

    温三农善于造一堆虚妄的高大上的身份头衔和组织，其核心团队的弟子们也继承了师傅这一江湖技艺，列举几个：

    乡建DAO、全球大学，发起人严晓辉。

    他自己都说不清楚，这俩组织到底是个什么东西？

    办大学历史最悠久的英美德等西方国家，都没做过什么全球大学，谁具备这样的资格？严晓辉这个没好好上过学的学渣，何德何能办全球大学？

    创造新概念、掌握话语权 #

    除了最有名的概念“三农问题专家”，他们还独创了类似“社会生态农业”的概念。

    中国最高等学府清华大学博士后、“全球青年领袖”石嫣，在她的导师温教授的指引下，将CSA的中文名改为“社会生态农业”，放弃了已经被媒体和公众广为接受的中文名称“社区支持农业”，“社区支持农业”是业界已经约定俗成用了很多年的，为什么这么做？这叫偷换概念。

    这就犹如中国计算机和互联网行业，经常有中国的专家和公司宣称中国自主研发了操作系统和芯片等等，其实，那只不过又拿了人家开源的代码和硬件，把外壳做了一个中文版汉化，打了一些补丁，普通公众并不懂自主研发的含义，以为又是中国人自己的发明创造，充满了民族自豪感。

    就是这样通过抄袭创造了一个个名词，这也叫创新？这就是祸国殃民的专家。

    总结 - 方法论 #

    你若是个有心人，想进入某个行业领域，带着虚心学习的态度，最好的办法是对权威祛魅。

    不必要去研究这些伪专家们到底做了什么，也不必去看他们写的成堆的书，大多数人都看不明白的，我们都知道很多专家的水平，东抄一点、西抄一点，拼凑在一起，堆砌复杂的术语，所有的概念都往里面装，里面思维逻辑混乱，无法自洽和自圆其说。

    你若不同意他们的观点，想找他们辩论，你也不可能赢，书是他们写的，话语解释权就他们嘴上说了算，他可以非常灵活地解释，甚至前后观点和立场颠倒。

    你只要稍微有点思维逻辑，去分析这些虚空的头衔，你就知道他们是个什么个货色、水份和品质了。这样你就能筛选出真伪，避免踩坑和浪费时间。
