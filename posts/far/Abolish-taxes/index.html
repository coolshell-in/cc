    乡村建设团队的文革情结

    2013-09-22 Atom.X

    温三农的黑历史 #

    这是我偶然接触到的一个案例。

    小毛驴回忆录

    赤脚医生 #

    大概在90年代，当时中国改革开放市场经济发展，而普遍农民收入无法供给生活成本，农民的税赋很重，当时温教授给《改革内参》杂志社任职，他去安徽做了系列调查采访，并组织当地的通讯记者站，手下有不少的当地记者合作写稿，为了让文章更有吸引力，他怂恿农民记者爆了很多猛料，一时间激起很大社会反响，他当然知道舆论和社会群众运动的威力和后果。

    但是他怂恿继续干，结果就发展到一群农民上访，甚至上街游行示威，安徽当地通讯社的农民记者懵懂无知，不知道如何处理，作为京城见过大世面的领导，温没有帮助这些农民记者，也没及时阻止和避免事态扩大，最终一批人进了监狱，其中一些人惨死。

    一位蹲了10年大牢的农民记者告诉我们，温当年怂恿他们去闹事，在关键时候他们到北京来找温，教授却躲起来了，并否认与事件的关联，狡猾的老狐狸躲过去了，但手下的那些记者和农民却很惨。

    这位农民记者，曾经也是一个农村的赤脚医生，靠着针灸和中医技能，在监狱里熬过最艰难的日子，最终活着出来了，然后人变得有点疯疯癫癫和精神恍惚。

    害得他坐牢，非常惨痛的经历。

    因此被刺和背叛，成了温三农的一个重要污点。

    您的问题涉及三个关键历史事件，时间点需准确区分。以下是基于权威史料和官方报道的澄清：

    事件背景 #

    一、中国全面取消农业税的时间 #

    • 法律废止时间：2006年1月1日
    • 法律依据：2005年12月29日，第十届全国人大常委会第十九次会议通过《关于废止〈中华人民共和国农业税条例〉的决定》，自2006年起施行。
    • 意义：终结了在中国延续2600多年的“皇粮国税”。

    ✅ 结论：全国范围内农业税于2006年正式取消。

    二、中国经济体制改革杂志社赴安徽调研的时间 #

    根据您此前提供的资料（该杂志社官网文章）：

    “中国经济体制改革杂志社调研组赴安徽省调研” 的时间为 2023年8月21日–24日。

    此次调研聚焦新时代安徽改革案例（如“大托管”、林长制、科技创新等），与农业税或农民抗议无关，属于常规性政策研究。

    ✅ 结论：该杂志社调查安徽是在2023年，内容为当代改革，非历史事件回溯。

    三、安徽农民因税费负担爆发大规模抗议的时间 #

    这指向的是 1990年代末至2000年代初的“三农危机”高峰期，其中最具代表性的是：

    ▶ 2000年–2001年：安徽多地农民抗税事件 #

    • 背景：
      农业税 + “三提五统” + 各种摊派导致农民实际税负远超法定水平，部分地区“一年收三年的税”。

    • 典型事件：

      • 2000年6月：安徽临泉县、阜阳市等地发生多起农民围堵乡镇政府、拒绝缴税事件；
      • 2001年：安徽利辛县、涡阳县等地因强制征收“农业特产税”引发冲突，有报道称农民与征税人员对峙。

    • 导火索：
      地方政府财政困难，将教育、医疗等支出转嫁到农民头上，“头税轻、二税重、三税是个无底洞”。

    • 结果：
      这些事件被《南方周末》《中国改革》等媒体曝光，成为推动农村税费改革的关键压力源。
      → 直接促成 2000年中央在安徽全省试点农村税费改革（全国首个试点省）。

    ✅ 结论：安徽农民因农业税及附加负担爆发大规模抗议主要集中在 2000–2001 年。

    四、三者时间线对照 #

    事件时间性质
    安徽农民抗税抗议2000–2001年社会危机，倒逼改革
    中央在安徽启动农村税费改革试点2000年政策响应
    全国全面取消农业税2006年1月1日制度终结
    中国经济体制改革杂志社调研安徽2023年8月新时代改革评估

    补充说明：安徽在农业税改革中的特殊地位 #

    • 2000年，中央选择安徽作为全国农村税费改革唯一试点省，正是因为其矛盾最尖锐、问题最典型；
    • 改革核心：“减轻、规范、稳定”——取消“三提五统”，农业税税率统一为 不超过常年产量的8.4%；
    • 安徽试点成功后，2002年扩大到20省，2003年全国推行，最终2006年彻底取消。

    正如时任国务院总理朱镕基所言：“安徽是农村税费改革的突破口。”

    总结回答 #

    • 中国取消农业税：2006年1月1日；
    • 中国经济体制改革杂志社调查安徽：2023年8月（调研新时代改革，非历史事件）；
    • 安徽农民因农业税爆发抗议：2000–2001年，直接推动全国农村税费改革启动。

    这三个时间点分别代表了危机爆发 → 政策试点 → 制度终结 → 后续评估的完整历史链条。
