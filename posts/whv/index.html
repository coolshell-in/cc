    劳动换食宿和打工度假网络计划

    2024-11-06 Atom.X

    解决年轻人向三农流动的方案 The Solution For Youth Flowing to Agri.Dept, Farmers & Rural Area

    项目概述 #

    本项目定义为涉农创业孵化、交流学习、社交平台，力图解决社会发展的首要因素 - “劳动力”，即首要解决引导“年轻人向三农流动”的难题。

    具体方法步骤为：

    • 初步引导年轻人作为志愿者/义工，通过劳动换食宿的方式进入三农（农业，农村，农民）领域探索体验；

    • 然后有一定比例的志愿者将愿意留在涉农领域做实习生，兼职或全职员工；

    • 最终这些人中又有极少部分开始创业，依托于本项目提供的综合性服务，直至达成目标；

    现在我们首先重点关注第一步的涉农志愿者平台开发和运营。

    招募义工的互联网平台有很多种，已经发展很成熟了，国内状况为鱼龙混杂良莠不齐，但至今尚未有一个质量较高且规模做得较大的平台，根据行业分类，主要有支教、农业、城市社区服务、公益组织、住宿交换、背包客。

    还有付费的游学或培训活动也叫义工招募，也就是你得花钱才能做义工。

    对标案例 #

    以农业类义工平台为例

    WWOOF #

    WWOOF（World Wide Opportunities on Organic Farms，世界有机农场机会组织）是一个国际性的网络，旨在连接希望体验有机农耕生活的人们（通常称为“志愿者”或“WWOOFers”）与全球各地的有机农场主。

    运作方式 #

    • 志愿者：通常是旅行者、学生、对可持续农业感兴趣的人，他们到有机农场帮忙劳动。
    • 工作内容：包括种植、收获、堆肥、照顾动物、修建围栏、做饭等日常农活。
    • 回报：农场主提供食宿（有时是简单的住宿和素食），但不支付工资。重点在于文化交流与学习有机农业知识。
    • 时间安排：通常每周工作4–6天，每天约4–6小时，具体依农场而定。

    如何参与 #

    1. 选择国家：每个国家有独立的 WWOOF 组织（如 WWOOF France、WWOOF Japan 等）。
    2. 注册会员：需支付年费（一般20–50美元），成为该国 WWOOF 会员后才能查看农场列表并联系农场主。
    3. 联系农场主：通过平台沟通，确认可接待时间、工作内容、语言要求等。
    4. 前往农场：自费前往，遵守约定，尊重当地文化和生活方式。

    适合人群 #

    • 预算有限的背包客
    • 对永续农业、生态村、自然生活感兴趣者
    • 想学习外语或跨文化交流的人
    • 希望慢下来、远离城市喧嚣的人

    注意事项 #

    • 不是打工度假（无薪），不能替代合法工作签证。
    • 劳动强度因农场而异，有些可能较辛苦。
    • 安全第一，建议提前了解农场评价，女性独自出行尤需谨慎。
    • 需具备基本责任感和适应能力。

    WWOOF是一个国际松散组织，不同于国际青年旅社的加盟模式，每个国家地区有独立域名的网站，其建站和运营方都不一样，填写申请资料的项目和费用均不同。

    据估计，每年进入中国的背包客wwoofer超过1万人，服务于2000家农场，按照目前200-300元人民币/年会员费计算，每年该类平台超过200万收入。

    但是该平台没有客服，仅仅提供农场列表和简单的申请表单，页面也并非全中文，有较多机器翻译的痕迹和错误，没有其他增值服务，用户体验较差。

    类似平台还有：

    • 互惠生
    • Couchsurfing
    • Helpx

    平台原型 #

    义工旅行源起自西方学生和年轻人的间隔年（gap year），他们在年轻的时候有时间停下学业和工作，去不同的地方和不同的行业领域，体验不同的人生，创造未来，学好玩好，工作好，生活好，当具备这样的状态和行动力，人生成功的定义就不受限于生计 ……

    义工的主要特征就是“劳动换食宿”，平台调性和本质为：年轻，阳光，国际化，科技，探索，好玩，低成本，吸引年轻人深度体验。

    相比之下，当今中国学生的课业普遍太沉重了，对于90年之前出生的学生而言，中学没有间隔年，大学想休学也不容易，上学的时候家庭和学校都不让恋爱，毕业就要忙着生计，离开学校没几年就被催着结婚买房生子，不敢轻易离职失业，害怕失去稳定的收入保障，更没时间停下来好好深度旅行（deep travel），体验不同的人生境遇，这种缺失导致我们从学校开始就成为社会化分工生产流水线的一颗螺丝钉，想象力和创造力就相对贫乏。

    我们依据以上多个国内外义工平台的案例，抽取其通用的原型/模型，描述如下。

    用户类型 #

    • Host 主人（免费注册）

    按照预先的约定，提供以下服务。

    吃饭/住宿

    业余休闲

    就业创业

    用户类型可以综合多样化，包括农场/ 工厂/ 旅舍/社区/ 学校/ 公益组织等等。

    • Guest 客人（付费注册）

    按照预先的约定，提供以下服务。

    工作3-4hrs/d

    年轻活力思维

    专业知识技能创意

    陪伴对话交流协作

    从 社会公众 → 客户（消费者）→ 义工 → 实习生/员工 → 创业者，这样的不同类型人群随着箭头所指的方向发展，其人员比例是递减的，因此人事发展是比较艰难的进程，我们今天有专业的新媒体和电商平台分别对标各种类型用户，但是还缺乏专业“义工”服务平台。

    收费方式 #

    目前按照国际主流模式，由Guest支付年度会员费，不同国家地区的价格不同，由于中国用户众多市场庞大，可以考虑从农场主一方收费，或者从双方两头各收一半，或完全取消收费，在用户数量达到一定程度，通过电商导购和广告等方式营收。

    时代背景 #

    一号文件 #

    中共中央 国务院关于实施乡村振兴战略的意见 http://www.gov.cn/zhengce/2018-02/04/content_5263807.htm

    到2018年，中国政府已经连续15年阐述对三农重视的决心和行动。

    • 双创

    中国政府，李克强总理以及新华社等媒体反复报道支持“大众创业，万众创新”。

    即使在改革开放近40年后的今天，许多年轻人还在想着公务员和企事业单位的铁饭碗，尽管政府一再提倡双创，因为“脑袋和人生体验”的贫穷“限制了想象力”，雷声大雨点小的双创效果并不明显，2018年全国大学应届毕业生再创新高达到800万，还有许多往届生处于失业状态。人力资源社会保障部的压力较大，这么多人该怎么安排？

    • 产业跨界

    城市产业资本纷纷跨界踏足农业领域，互联网巨头时常到农村刷墙宣布要投资农业。

    城市 vs 农村 #

    城市与农村各有优势劣势，对于今日的农村而言，已非同往昔，房屋宽阔，食物富足且相对品质更好，利用这个特别优势，刚好可以做点什么。

    维度城市农村
    优势
    经济与就业经济发达，就业机会多拥有闲置土地和剩余劳动力
    居住与空间——房屋宽阔，居住空间充裕
    环境与资源——自然生态环境良好，物产丰富且品质较高
    文化与教育教育资源优质，文化生活丰富——
    劣势
    社会问题“城市病”突出（拥堵、快节奏等）年轻人外流严重，人口结构失衡
    居住成本房价高，生活成本压力大——
    工作与心理工作压力大，竞争激烈就业机会少，收入渠道有限
    健康与安全食品安全风险较高，环境污染较严重——
    教育与文化——教育资源匮乏，文化娱乐活动稀缺
    人口与陪伴——留守老人、儿童、妇女普遍，缺乏陪伴与照护

    • 农村之路

    中国的农村就像一块巨大的海绵水池，当城市发展需要时，从农村源源不断挤出水分供给城市和各行业发展所需，最重要的农村资源是培养了10多年的年轻人力资源。当城市不堪重负出现产业和人力资源饱和时，或者因为农村建设需要时，又从城市排出大量人力资源，让他们流回农村，这个巨大的海绵水池有这样的泄洪抗旱能力。

    近代历史上出现过的“知识青年上山下乡”运动，有其特定时代背景，其根源也在于我们三农的任务太重太艰巨，离现代化的距离太大太远，单靠农村的内因是不可能实现的，没有一些非常举措，诸如大力动员城市的力量各行各业的支援等，是不可能实现的。

    20世纪50-70年代，我们维持了领土边防和国家安全，以及基本的工业化；80年代至今花了30多年，以城市和工业发展的战略优先，因此我们至今还没有全面完成三农现代化，历史发展到支持三农全面建设正当其时。

    平台目标 #

    当政策，资本和市场具足，且中国越来越多的农村地区已经在基础设施（水，电，网，路，快递）等方面条件日益改善，许多地区不比城市差，如果三农整体现状还没有较大改善，只能说明当下最缺现代三农人才！！！

    增值服务 #

    目前wwoof只有简单的线上自助申请对接双方用户的功能，用户量将倍增的关键是增值服务，包括：签证，保险，租车，交通补贴，免年费，多语言版本等。

    还有包括更重要的基于三农的创业孵化服务。

    创业孵化 #

    鼓励大学生和社会青年通过平台组团创业，提供创业孵化所需要的培训，资金，甚至创业园后勤保障服务等。

    农创客 - 乡村沃野，青年农创客起新潮

    最近5年城市科技园从事双创尚未成熟，而各地农村地区又开始做“双创”，换了一个名字叫“星创天地，农创客”，其本质是一样的，盖了一堆楼，却没有好好利用，依然是地产模式，政府买单让我们去做农业电商培训，只有培训机构和运营商赚到钱。

    一批比较精明的电商运营服务商在各地开始接管政府主导的双创园，将以前山寨的“淘宝城”招牌换成了“星创天地”，又套了一笔补助款，但创业效果并不明显。

    因此我经历不同的地区和案例之后，得出一个结论是，可能需要一些公益组织，以及尚没有明确利益动机的志愿者去参与三农的双创，或许会产生不一样的结果，因为他们没有强烈利益刺激，社会理想抱负和名声大于赚钱的想法。

    社会效益 #

    为了说明平台如何运作，列举该现实案例

    • 冰花男孩

    由于地处偏远，需要大量志愿者协助当地至少3个方面的建设：

    经济

    扶贫：优势农业资源，例如邵通苹果

    机构：对接销售渠道，例如e农计划，诚实善粮等社会企业，以及其他常规流通渠道；

    功能：帮助偏远农村地区销售优质农产品，提高贫困农民收入，让爸爸妈妈回家，让山村的孩子不再留守。

    教育

    支教：山村孩子的守护天使

    机构：对接支教公益组织，例如天使支教，以及其他自然教育（非应试教育）资源；

    功能：为乡村儿童提供长期成长陪伴服务，进行教育创新探索（创造力培养与归属感建立）的公益项目。

    环境

    环保：减少农药化肥污染

    机构：对接环保技术，例如云南思力生态替代技术中心，以及其他环保产业技术；

    动作：提高农业劳动生产效率和产品品质，保护环境，不以牺牲环境为代价换取短期的发展利益。

    我们可以通过本平台，为该地区提供源源不断的优质人力资源保障，以及创业孵化，公益支持。

    其他案例 #

    • 朴和素

    位于杭州富阳龙门镇，该家庭农场农场面积很小，从来都不做商业推广。房子小而且破，冬天漏风，条件不好，但依然有不少外国志愿者申请，经常要拒绝掉很多人，几乎每个周末都有各地的朋友来探访。

    • 任之堂

    位于湖北十堰，完全通过志愿者改造抛荒20年的乡村，变为网友热捧的中医村。通过中医诊所、任之堂养心山庄，弘扬中医文化，宣传正确养生理念！

    • 无畏护生园

    该庇护农场位于河南焦作武陟，1200多亩，生活着60多种动物，于2014年创办，焦作市动物保护救助协会筹建，河南泰鑫农业有限公司捐赠土地使用，旨在保护救助被遗弃和被暴力伤害的流离失所的无助动物。

    目前该基地的资金主要来源于释妙无法师自己的积蓄，以及来自各处的善人捐款；同时吸纳许多网络上对农场动物保护感兴趣，或想尽自己一份力的志愿者团队。

    劣势弊端 #

    以常见招募义工的民宿为例，因为经营单位很小且多元化，非标准，导致管理成本高，效率低，营收少，很多难以为继。

    所以，民宿的花花绿绿是投资者老板们的情怀，因为他们喜欢，并代表一种集体的情怀，把自己有限的青春和资产奉献给广大的穷游族们，不按市场规律操作，注定会是一场乌托邦的梦想，但是依然会前仆后继，后浪推前浪地扑向沙滩。

    为什么大型连锁酒店不愿意招募义工？因为有非常标准化的职业培训管理体系，义工反而是一种高成本的操作方式。

    关键点是将服务业的管理标准化，作为软件平台的核心功能，在招募前，中，后进行帮扶管理，但是不能与一般的住宿预定平台airbnb类重复，应摘取最核心功能即可。

    对于任何一种类型的服务业，其最核心的抽象模型标准都是一样的，具体操作不同，可以因地制宜，因人而异，与人机交互设计的基本原则相同：

    1. 安全卫生；
    2. 用户体验
    3. 便捷高效；
    4. 易于管理；
