    三农 #

    “三农”（农业、农村、农民）作为一个固定术语，最早在2000年3月由时任湖北省监利县棋盘乡党委书记李昌平上书朱镕基总理时正式提出。他在信中写道：“农民真苦，农村真穷，农业真危险”，引发全国关注。

    此后，“三农问题”迅速被主流媒体和政策文件引用，并在2001年以后成为中央一号文件及政府工作报告中的高频词。

    自2004年起，中共中央连续多年发布以“三农”为主题的中央一号文件，标志着“三农问题”正式进入国家核心政策议程。

    乡村建设 #

    • 早期起源（1930年代）：
      “乡村建设”最早可追溯至1930年代，以梁漱溟在山东邹平（1931–1937）、晏阳初在河北定县等地开展的乡村建设运动为代表。这是中国近代第一次系统性的乡村改良实验。

    • 当代复兴：
      现代意义上的“乡村建设”话语在21世纪初重新兴起，主要由学者如温铁军等人推动。

      • 2003年，温铁军联合《中国改革》杂志团队在河北省定州市翟城村重启乡村建设试验，创建了晏阳初乡村建设学院；
      • 同年，他获得CCTV中国经济年度人物奖，被吴敬琏称为“中国农民的代言人”，“乡村建设”由此进入主流媒体视野。

    • 政策升级：

      • 2005年，党的十六届五中全会正式提出“建设社会主义新农村”；
      • 2017年，党的十九大提出“乡村振兴战略”，“乡村建设”内涵进一步扩展。

    温铁军 #

    温铁军的三农团队历史悠久，还做了很多相关工作，包括：

    • 《中国改革》杂志社

    • 2004年，中国人民大学 - 农业和农村发展学院 - 乡村建设中心（901办公室），是温三农一切工作的指挥中心；担任人大农发院的院长，利用职务之便，持续开展了三农相关工作；

    • 2004年，在北京海淀区温泉镇，成立北京梁漱溟乡村建设中心，其前身可追溯至2001年《中国改革》杂志社的大学生支农调研项目部，后隶属于中国人民大学乡村建设中心的分支机构；

    • 2007年， 晏阳初乡村建设学院，被定州当地有关部门和村民强行关闭，温三农核心团队连夜撤走；

    • 2008年，在北京海淀区温泉镇凤凰岭，成立小毛驴市民农园（农场），由温三农核心团队运营管理；

    有密切业务关联的组织包括

    和平妇女

    香港岭南大学

    新工人，打工者之家，皮村。

    资金来源：

    • 公益基金会：福特基金会、香港乐施会、香港嘉道理基金会、等10多家基金会；
    • 学校和政府：通过人民大学关联的单位，乡村建设项目地的地方政府和村镇集体企业，获得部分项目资金；
    • 私人捐赠：通过温三农个人关系，有极少量的捐赠；

    三农问题专家 #

    大概从2000年开始，温教授是主流媒体上的明星，确立了“三农问题专家”和“乡村建设”的身份标签，他就是这2个专有名词（概念）的所有人。

    顺应时代潮流，现在2025年改叫“乡村振兴专家”，他们又换了块牌匾，成立了多处“乡村振兴研究院”（接着忽悠）。

    温三农被媒体包装成“用脚做学问”的泥腿子专家，也就是说，他一直亲身在农村一线的田间地头实践，他是实践派和实干家，区别于其他的只会讲纯理论的知识分子。

    但是当被问到：“你们的乡村建设具体实践是做什么？”，他又说从事“乡村试验”。他总是在这两个概念（建设、试验）之间倒腾来倒腾去，确实看起来像是做化学实验时，拿两个量杯给开水降温的样子。

    然后继续追问：

    • “试验”能算是“建设”吗？
    • 搞了20多年乡村建设，实践成果在哪里？
    • 为什么一直在做试验？没完没了地试验？

    长期关注三农领域的同学就好奇了，三农问题专家们，搞了20多年乡村建设和试验，无一成功案例，依然还在做乡村试验。

    为什么温三农20多年来一直在做“乡村建设和试验”？却毫无进展和成功案例？世上哪有这样的奇葩试验？

    谁都无法回答以上这些问题，面对媒体时，他们一直在圆滑地处理这些发问，似乎“那不是他们的职责”。

    这几句追问，让人幡然醒悟，就验证了温三农无论是建设（实践）或试验（理论）从未有过成果，有的只是挂着虚高教授博导专家的名头。

    他们耗费国家社会大量的资源，在实验室里捣腾一堆烂东西，专家所到之处，都是一地鸡毛和狗血剧情，还动辄开新闻发布会，要在全国各地（忽悠）推广（实现量产）经验，这不就是公然的诈骗和胡说八道么？

    若非体制内，享用着编制福利，这帮教授和博士肯定会饿死的，因为没有任何社会贡献产出。花着纳税人和教育基金的钱，为什么可以一直花钱，却没任何试验成果产出？这是谁的问题？深究起来，这又是教育体系腐败和官僚主义问题。

    温三农的乡村建设和乡村试验具体是在做什么？真实动机是什么？

    乡村调研 #

    通过梁漱溟乡村建设中心的官网页面介绍，说得更客观具体点，温三农的乡村建设和乡村试验，其实一直在做“乡村调研”，调研才是他的主业。

    他们总是找一群没有就业的8流大学的学生，这些和乡村社会格格不入的学生，下放到农村能做什么呢？他们能解决什么问题呢？没有力气，能种地吗？能养殖吗？不懂技术，更不会水电管道维修。

    学生们整天和农民混在一起，辅助干点农活外只能聊天，打探农民家隐私，写到调研报告中。

    于是“乡村实验”具体怎么做，就有理由了，一直没玩没了地做调研，让学生找农民家了解情况，写一堆各种报告和书，这就是作为温三农的学术成果，完全没有任何解决方案。

    乡村建设的成果是什么？ #

    乡村属于社会范畴概念，社会建设主要有物质和精神文明两个方面，乡村建设也应该包括至少这两部分。

    在他庞大的组织体系中，由他的学生亲自运营，或经过他们培训指导的项目，非常非常多，好结局的有哪些？成果呢？他无法列举。

    这说明其动机根本就不是乡村建设，自然就不会有相应的（物质和精神文明）成果。

    物质文明 #

    温三农团队从未做过一个乡村建设的成功案例，因此他喜欢推荐别人做的经典案例，缺了案例，否则这学术报告怎么写呢？

    神奇的是，只要贴了温三农牌子的别人的案例，也基本大量都烂尾了，谁和温扯上关系，谁就倒霉破产。

    这些集团全部都在随后几年内破产倒闭。

    2011年南南论坛，我受温教授团队指派服务10多国家参会的专家学者。

    参观访问过其中一些知名的项目地，包括并不限于以下案例：

    1. 江苏江阴华西村；
    2. 浙江登峰集团 浦阳镇桃北新村,云都美浓小镇；
    3. 北京小毛驴市民农园；

    当时温教授让我们带了十多个国家的考察团去看这些案例，为了彰显中国社会主义新农村建设的成功案例，对这几个案例大吹特吹，结果呢，没几年都宣告破产。

    知道更多案例的朋友，欢迎提供。

    精神文明 #

    空的，完全没有，曾经我给他们做网站，系统整理过他们的图片和文案，这个项目由温三农团队的秀才黄志友负责，他只给我几张北方大妈在晒谷场上跳秧歌的图片，这张图让我印象深刻，因为拍摄年代久远，农民身上穿的衣服都是打补丁的，可能是那种饥荒年代，或改革开放之前拍的。

    所以，我当时就起了疑心，乡村建设做了什么精神文明工作吗？没有，完全没有。
