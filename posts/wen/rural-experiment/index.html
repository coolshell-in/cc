    三农问题 #

    “三农”（农业、农村、农民）作为一个固定术语，最早在2000年3月由时任湖北省监利县棋盘乡党委书记李昌平上书朱镕基总理时正式提出。他在信中写道：“农民真苦，农村真穷，农业真危险”，引发全国关注。

    此后，“三农问题”迅速被主流媒体和政策文件引用，并在2001年以后成为中央一号文件及政府工作报告中的高频词。

    自2004年起，中共中央连续多年发布以“三农”为主题的中央一号文件，标志着“三农问题”正式进入国家核心政策议程。

    乡村建设 #

    • 早期起源（1930年代）：
      “乡村建设”最早可追溯至1930年代，以梁漱溟在山东邹平（1931–1937）、晏阳初在河北定县等地开展的乡村建设运动为代表。这是中国近代第一次系统性的乡村改良实验。

    • 当代复兴：
      现代意义上的“乡村建设”话语在21世纪初重新兴起，主要由学者如温铁军等人推动。

      • 2003年，温铁军联合《中国改革》杂志团队在河北省定州市翟城村重启乡村建设试验，创建了晏阳初乡村建设学院；
      • 同年，他获得CCTV中国经济年度人物奖，被吴敬琏称为“中国农民的代言人”，“乡村建设”由此进入主流媒体视野。

    • 政策升级：

      • 2005年，党的十六届五中全会正式提出“建设社会主义新农村”；
      • 2017年，党的十九大提出“乡村振兴战略”，“乡村建设”内涵进一步扩展。

    #

    温铁军的三农团队历史悠久，做了很多相关工作，包括：

    • 《中国改革》杂志社

    • 2004年，中国人民大学 - 农业和农村发展学院 - 乡村建设中心（901办公室），是温三农一切工作的指挥中心；担任人大农发院的院长，利用职务之便，持续开展了三农相关工作；

    • 2004年，在北京海淀区温泉镇，成立北京梁漱溟乡村建设中心，其前身可追溯至2001年《中国改革》杂志社的大学生支农调研项目部，后隶属于中国人民大学乡村建设中心的分支机构；

    • 2007年， 晏阳初乡村建设学院，被定州当地有关部门和村民强行关闭，温三农核心团队连夜撤走；

    • 2008年，在北京海淀区温泉镇凤凰岭，成立小毛驴市民农园（农场），由温三农核心团队运营管理；

    • 私人捐赠：通过温教授的个人关系，有极少量的捐赠；

    三农问题专家 #

    大概从2000年开始，温教授是主流媒体上的明星，确立了“三农问题专家”和“乡村建设”的身份标签，这2个专有名词（概念）指代的就是他。

    顺应时代潮流，2025年现在改叫“乡村振兴专家”，他们又换了块牌匾，成立了多处“乡村振兴研究院”（接着忽悠）。

    温三农被媒体包装成“用脚做学问”的泥腿子专家，也就是说，他一直亲身在农村一线的田间地头实践，他是实践派和实干家，区别于其他的只会讲纯理论的知识分子。

    但是当被问到：“你们的乡村建设具体实践是做什么？”，他又说从事“乡村试验”。他总是在这两个概念（建设、试验）之间倒腾来倒腾去，确实看起来像是做化学实验时，拿两个量杯给开水降温的样子。

    然后继续追问：

    • “试验”能算是“建设”吗？
    • 搞了20多年乡村建设，实践成果在哪里？
    • 为什么一直在做试验？没完没了地试验？

    长期关注三农领域的同学就好奇了，三农问题专家们，搞了20多年乡村建设和试验，无一成功案例，依然还在做乡村试验。

    为什么温三农20多年来一直在做“乡村建设和试验”？却毫无进展和成功案例？世上哪有这样的奇葩试验？

    谁都无法回答以上这些问题，面对媒体时，他们一直在圆滑地处理这些发问，似乎“那不是他们的职责”。

    这几句追问，让人幡然醒悟，就验证了温三农无论是建设（实践）或试验（理论）从未有过成果，有的只是挂着虚高教授博导专家的名头。

    他们耗费国家社会大量的资源，在实验室里捣腾一堆烂东西，专家所到之处，都是一地鸡毛和狗血剧情，还动辄开新闻发布会，要在全国各地（忽悠）推广（实现量产）经验，这不就是公然的诈骗和胡说八道么？

    若非体制内，享用着编制福利，这帮教授和博士肯定会饿死的，因为没有任何社会贡献产出。花着纳税人和教育基金的钱，为什么可以一直花钱，却没任何试验成果产出？这是谁的问题？深究起来，这又是教育体系腐败和官僚主义问题。

    温三农的乡村建设和乡村试验具体是在做什么？真实动机是什么？

    乡村调研 #

    通过梁漱溟乡村建设中心的页面介绍，终于搞清楚了乡村建设和乡村试验的主业，说得更客观具体点，温三农团队其实一直在做“乡村调研”，调研才是他的主业。

    他们总是找一群没有就业的大学生，几乎都来自不怎么好就业的大学和专业，这些和乡村社会格格不入的学生，下放到农村能做什么呢？能解决什么问题呢？不懂技术，没有力气，不会种地，也不会养殖，更不会水电管道维修。

    学生们整天和农民混在一起，辅助干点农活外，就是聊天，让学生找农民家了解情况，打探农民家隐私，写到调研报告中。

    于是“乡村实验”具体怎么做，就有理由了，一直没完没了地做调研，写一堆各种报告和书，这就是作为温三农的学术成果。

    乡村建设成果是什么？ #

    乡村属于社会范畴概念，社会建设主要有物质和精神文明两个方面，乡村建设也应该包括至少这两部分。

    在他庞大的组织体系中，由他的学生亲自运营，或经过他们培训指导的项目点，几乎分布在中国各个省区，非常非常多，好结局的有哪些？成果呢？他无法列举。

    这说明其动机根本就不是乡村建设，自然就不会有相应的（物质和精神文明）成果。

    物质文明 #

    没有，完全没有

    三农专家们做的项目都是永远奶不大的孩子，如果哪天政策补贴断奶，他们就活不下去了。

    他们不会赚钱，更不会帮农民赚钱，凡是赚钱的事与他们无关，我们都知道三农乡村建设一直都很穷困潦倒。

    你绝对找不到任何一个他们做过的项目是不依赖公益基金、政策补贴和义工的。而且他们亲自干的项目几乎没有不烂尾的。

    物质文明这一块就直接忽略了。

    精神文明 #

    没有，完全没有。

    我曾经给温教授旗下的多个三农发展项目做网站，这个任务由温三农乡村建设核心的秀才 - 黄志友负责，他是小毛驴的会员内刊总编辑，负责对外的一切宣传事务。

    当时为了找图片和文案填充网页，我找黄志友要素材，他给了我一个网盘，里面保存了几乎温三农发展的整个历史，因此我算是系统地整理过温三农资料库的少数几个人之一。

    但是翻来覆去地找，他只给两我几张北方大妈在晒谷场上跳秧歌的图片，这张图让我印象深刻，因为拍摄年代久远，农民身上穿的衣服都是打补丁的，可能是那种饥荒年代，或改革开放之前拍的。

    所以，我当时就起了疑心，温三农的乡村建设做了什么精神文明工作吗？

    在扫描黄志友提供的三农工作的网盘过程中，虽然精神文明相关的网页素材没有找到，却意外发现了不少特殊的视频文件名，和黄老师的姓名一样的颜色，可见当时乡村建设的几个老光棍们很饥渴，饥不择食了，拿带日本小姐姐的视频为贫乏的大脑神经充饥，这也温三农团队的乡村精神文明建设。

    网红烂尾工程 #

    温三农团队虽然从未做过一个乡村建设的成功案例，但他会推荐别人做的经典案例，否则缺了案例，这学术报告和书怎么写呢？

    他们特喜欢那种名头特别大，特别响亮的网红项目。

    神奇的是，只要贴了温三农牌子（别人做的，与他无关）的三农先进案例，有不少都烂尾的，似乎谁和温扯上关系，谁就可能要倒霉了。

    2011年冬天 南南论坛(SSFS)，我为考察团提供会务活动服务，陪同来自12个国家的学者专家，调研访问中国东部（江浙沪）地区。

    其中一些知名集团，在随后几年内破产倒闭，列举其中部分：

    江苏江阴华西村 #

    被誉为“天下第一村”，是中国农村集体经济、共同富裕和乡村工业化最著名的典范之一。自20世纪60年代起，在老书记吴仁宝的带领下，从一个贫困村崛起为全国首富村，其发展模式被称为“华西模式”。

    我有幸亲自和老书记握过手，他的方言接近上海话，我们都听不懂，他的孙女给我们做翻译。

    我们还住了由村庄集体建造的全球最高建筑 - 华西龙希国际大酒店（Longxi International Hotel）74层328米，因为村子太小，即使兼并周围几个村的土地，还是无法满足发展需求，于是往高处盖楼，也叫华西村增地大厦。

    在最大的2000平方米宴会厅用餐，可容纳100桌以上，约1000–1200人同时就餐，这是江阴及周边地区规模最大的单体无柱宴会厅之一，常用于婚宴、企业年会、政府接待等；

    迎接各国专家和游客的还有一场综艺节目，同时可以用8种语言做开幕播报，现场氛围堪比春晚，极其精彩纷呈和土豪气质，把各国专家都看懵了，中国闷声发大财，怎么可能还是发展中国家？

    但是，2016年当我在网易养猪的时候，某一天刷到华西村债务危机的新闻，距离上次参观考察才过去3年多。

    然后2018年危机全面爆发，峰值负债约400亿元人民币，负债率高达70%，这不是一天发生，而是早在我们去参观之前就埋下隐患，作为经济学家，难道温教授看不出来？

    云都美浓小镇 #

    项目地位于杭州市萧山区的浦阳镇桃北新村，是中国第一批美丽乡村和千万工程示范点。浙江的乡村建设水平都很高，创造了很多中国第一，包括：“绿水青山，就是金山银山”（两山）、美丽乡村、“千村示范、万村整治”工程（千万工程）等。

    打造该小镇的是浙江登峰交通集团，这家成立于1997年的浙江本土企业，曾涉足公路桥梁、市政园林、房地产等多个领域，但因2015年地产项目资金链断裂，成为浙江省内具有代表性的大型民企债务风险案例。

    当我2013年还在淘宝农业部门上班，某一天刷到登峰集团债务危机，已进入破产清算程序，至今未恢复经营。

    北京小毛驴市民农园 #

    有机/生态农业和三农发展领域的人都知道这家网红农场，自从2012年石嫣分家并建立CSA分享收获农场，小毛驴就逐渐退出历史舞台的聚光灯，情形每况愈下，我当年给农场做的网页也在几年后逐渐消失了；

    温教授让我们带外国专家去看这些案例，为了彰显中国社会主义新农村建设的巨大成功，对这几个案例大吹特吹，结果呢，没几年纷纷宣告破产。

    其他还有几条线路，包括华北和西南路线，我没有去，不在此列举，必定也有一堆他们主推的烂尾网红项目。知道更多温三农细节的朋友，欢迎提供他们推过的烂尾网红案例。
