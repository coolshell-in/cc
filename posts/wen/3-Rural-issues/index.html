    乡村建设、三农问题专家的历史源流考据

    2010-03-15 Atom.X

    三农问题 #

    “三农”（农业、农村、农民）作为一个固定术语，最早在2000年3月由时任湖北省监利县棋盘乡党委书记李昌平上书朱镕基总理时正式提出。他在信中写道：“农民真苦，农村真穷，农业真危险”，引发全国关注。

    此后，“三农问题”迅速被主流媒体和政策文件引用，并在2001年以后成为中央一号文件及政府工作报告中的高频词。

    自2004年起，中共中央连续多年发布以“三农”为主题的中央一号文件，标志着“三农问题”正式进入国家核心政策议程。

    乡村建设 #

    • 早期起源（1930年代）：
      “乡村建设”最早可追溯至1930年代，以梁漱溟（1931–1937）在山东邹平、晏阳初（1926–1937）在河北定县等地，开展的乡村建设运动为代表。这是中国近代第一次系统性的乡村改良实验。

    • 当代复兴：
      现代意义上的“乡村建设”话语在21世纪初重新兴起，主要由学者如温铁军等人推动。

      • 2003年，温铁军携带《中国改革》杂志团队在河北省定州市翟城村重启乡村建设试验，创建了晏阳初乡村建设学院；
      • 同年，他获得CCTV中国经济年度人物奖，被吴敬琏称为“中国农民的代言人”，“乡村建设”由此进入主流媒体视野。

    • 政策升级：

      • 2005年，党的十六届五中全会正式提出“建设社会主义新农村”；
      • 2017年，党的十九大提出“乡村振兴战略”。

    温铁军 #

    截至2025年12月的权威媒体报道、机构官网及个人公开资料，其常见头衔包括：

    一、学术与教育机构头衔 #

    1. 中国人民大学学术委员会副主任
      （原为中国人民大学农业与农村发展学院院长）

      人民大学的身份这是温教授最重要的身份标签，约2010年代后期已正式退休，但因影响力持续，仍保留多项荣誉性职务。

    2. 西南大学中国乡村建设学院执行院长
      （该学院由温铁军团队主导，聚焦乡村振兴与生态农业）

    3. 福建农林大学新农村发展研究院特聘教授 / 乡村振兴研究院院长
      （近年深度参与福建“生态省”与集体林权改革研究）

    4. 中国人民大学可持续发展高等研究院院长
      （推动“生态文明”与“去增长”理论本土化）

    二、社会职务与公益身份 #

    1. 北京共仁公益基金会名誉理事长
      （该基金会支持全国CSA社区支持农业、合作社与青年返乡项目）

    2. 国仁乡建社会企业联盟首席顾问
      （推动“社会化小农经济”与城乡融合实验）

    3. 中国农业大学人文与发展学院兼职教授
      （参与“发展研究”与“南南合作”课程）

    三、媒体常用标签（非正式但高频出现） #

    1. 著名“三农”问题专家
      （几乎所有主流媒体如新华社、央视、《人民日报》对其报道均使用此称谓）

    2. 乡村振兴战略研究专家
      （政策解读类节目常用头衔）

    3. “人民经济学”倡导者
      （近年因其批判新自由主义、强调“成本转嫁论”“第三资产池”等理论而被赋予的学术标签）

    4. “用脚做学问”的实践型学者
      （源自其本人名言：“真正符合中国实际的学问是从泥土里长出来的”）

    四、曾用或历史头衔 #

    • 《中国改革》杂志社社长兼总编辑（2000年代）
    • 国务院学位委员会学科评议组成员（已卸任）
    • 中央党校、国家行政学院等干部培训特聘讲师

    五、网络平台认证信息示例 #

    • 今日头条主页认证（2025年）：

      “著名三农学者｜北京共仁公益基金会名誉理事长”

    • 微博/抖音/B站简介：

      “中国人民大学教授，长期从事‘三农’与乡村建设研究”

    三农组织社团 #

    温铁军的乡村建设和三农团队的历史悠久，主要包括：

    • 《中国改革》杂志社

    • 2004年，中国人民大学 - 农业和农村发展学院 - 乡村建设中心（901办公室），是温三农一切工作的指挥中心；担任人大农发院的院长，利用职务之便，持续开展了三农相关工作；

    • 2004年，在北京海淀区温泉镇，成立北京梁漱溟乡村建设中心，其前身可追溯至2001年《中国改革》杂志社的大学生支农调研项目部，后隶属于中国人民大学乡村建设中心的分支机构；

    • 2007年， 晏阳初乡村建设学院，被定州当地有关部门和村民强行关闭，温三农核心团队连夜撤走；

    • 2008年，在北京海淀区温泉镇凤凰岭，成立小毛驴市民农园（农场），由温三农核心团队运营管理；

    关联组织 #

    受温三农领导，或与其有密切关联的组织包括

    温三农旗下的学生有两大派系

    • 学院派，以石嫣为代表，是正规的院校出来，在高等学府深造过，至少出过国，能讲点英语；
    • 野草派，以严晓辉为代表，较早一批跟着温教授搞大学生下乡调查，没有好好上过学，野草一样生长出来；

    和平妇女

    香港岭南大学

    新工人，打工者，皮村工友之家。

    资金来源：

    • 公益基金会：福特基金会、香港乐施会、香港嘉道理基金会、等10多家基金会；
    • 学校和政府：通过人民大学关联的单位，乡村建设项目地的地方政府和村镇集体企业，获得部分项目资金；
